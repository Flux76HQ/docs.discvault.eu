## Procedur

1. **Öppna:** Bibliotek → Sök
2. **Konfigurera:** Bibliotek → Filter → Format / Genre / Grupp
3. **Jämför:** Bibliotek → Vy → Affischer / Lista
4. **Testa:** Titel → Detaljer → Bevakningslista / Sedd
5. **Öppna:** Inställningar → Preferenser → Collectors → Collectors mode / Merge multiple editions as one title
6. **Konfigurera:** Collectors Mode är av som standard och visar avancerade containerfunktioner; `merge_editions_as_title` är en separat inställning som också är av som standard
7. **Testa:** när `merge_editions_as_title` är aktiverat visas flera fysiska versioner som en biblioteksruta; öppna rutan för att se varje utgåva
8. **Jämför:** ett box-set är en fysisk handelsprodukt med egen streckkod; filmerna förblir individuella titlar länkade som direkta filmmedlemmar
9. **Jämför:** ett vault är en användarskapad hyll-, förvaringsbox- eller platsgrupp med direkta filmmedlemmar och utan betydelse som handelsprodukt
10. **Jämför:** en collection är en tematisk grupp vars ordnade objekt kan vara filmer, box-set, vaults eller andra collections, så nästling stöds
11. **Testa:** att stänga av Collectors Mode döljer containervisningen men raderar inte containrar eller medlemmar
