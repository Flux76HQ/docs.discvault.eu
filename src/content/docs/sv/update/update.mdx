## Procedur

1. **Kontrollera:** nya installationer använder DiscVault v26 `:latest` eller `:beta`; den frysta avbildningen `:legacy` är endast för en befintlig installation av föregående generation som ännu inte är redo att migrera
2. **Säkerhetskopiera:** matchande säkerhetskopieuppsättning
3. **Välj:** Endast Legacy-grenen: efter säkerhetskopieringen ändrar du bara avbildningstaggen för den befintliga gamla installationen till `:legacy`, hämtar och återskapar den med oförändrad topologi, kontrollerar samlingen och stoppar; växla inte den befintliga installationen till v26-topologin på den här sidan
4. **Öppna:** [Uppdatering och migrering: Migrering till DiscVault 26](/sv/update/migration/) · DiscVault 26
5. **Bevara:** behåll föregående generations volymer, miljövariabler, portar och datasökväg oförändrade; ersätt aldrig en avbildning i v26 PostgreSQL Compose-topologin med `:legacy`
6. **Verifiera:** `DiscVault v26` · `PREVIOUS_IMAGE=@sha256:…`
7. **Välj:** `DiscVault v26` · `DISCVAULT_IMAGE` · `:latest` `:beta`
8. **Kör:** `DiscVault v26` · `--force-recreate`
9. **Jämför:** `DiscVault v26` · avsett image-digest körs, hälsan lyckas och ägarinloggning, biblioteksantal och ett medieprov godkänns

### Kommandon och inställningar

#### DiscVault v26 – stabil

```bash
export DISCVAULT_IMAGE=ghcr.io/helmerznl/discvault:latest
```

#### DiscVault v26 – beta

```bash
export DISCVAULT_IMAGE=ghcr.io/helmerznl/discvault:beta
```

#### Båda kanalerna

```bash
set -euo pipefail
test -n "$DISCVAULT_IMAGE"
case "$DISCVAULT_IMAGE" in
  ghcr.io/helmerznl/discvault:latest|ghcr.io/helmerznl/discvault:beta) ;;
  *) printf '%s\n' "$DISCVAULT_IMAGE" >&2; exit 64 ;;
esac
cd /opt/discvault
ENV_FILE=/opt/discvault/.env
ENV_TMP="$(mktemp "$ENV_FILE.update.XXXXXX")"
trap 'rm -f "$ENV_TMP"' EXIT
awk -v image="$DISCVAULT_IMAGE" '
  /^DISCVAULT_IMAGE=/ { print "DISCVAULT_IMAGE=" image; found=1; next }
  { print }
  END { if (!found) exit 42 }
' "$ENV_FILE" > "$ENV_TMP"
chmod 0600 "$ENV_TMP"
docker compose --env-file "$ENV_TMP" -p discvault -f compose.yaml config --quiet
docker compose --env-file "$ENV_TMP" -p discvault -f compose.yaml pull next-api next-worker next-mcp
docker compose --env-file "$ENV_TMP" -p discvault -f compose.yaml up -d --force-recreate
docker compose --env-file "$ENV_TMP" -p discvault -f compose.yaml ps
curl --fail http://localhost:6080/api/next/health
mv "$ENV_TMP" "$ENV_FILE"
trap - EXIT
```
