## Förutsättningar

- ägar- eller administratörssession
- leverantörsuppgift lagrad som hemlighet

## Procedur

1. **Kontrollera:** `GET /api/next/plugins/registry`
2. **Konfigurera:** Administration → Insticksprogram → Leverantör → Konfigurera · leverantörsuppgift lagrad som hemlighet
3. **Testa:** Administration → Insticksprogram → Hälsokontroll · Administration → Insticksprogram → Upptäck utan att spara · `dryRun`
4. **Anteckna:** `secret app_settings`

### Kommandon och inställningar

#### Båda kanalerna

```bash
curl --fail http://localhost:6080/api/next/plugins/registry
curl --fail http://localhost:6080/api/next/metadata/plugins
```
