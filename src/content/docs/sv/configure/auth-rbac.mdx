1. **Öppna:** Administration → Säkerhet → Aktivera autentisering
2. **Skapa:** Administration → Användare → Skapa 48-timmarsinbjudan
3. **Konfigurera:** Administration → Roller → Grundläggande / Avancerat
4. **Testa:** `48 h`
