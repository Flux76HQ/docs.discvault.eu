## Vereisten

- aangemeld account met de vereiste rechten
- `watchlist`

## Procedure

1. **Openen:** Bibliotheek → Zoeken
2. **Configureren:** Bibliotheek → Filter → Formaat / Genre / Groep
3. **Vergelijken:** Bibliotheek → Weergave → Posters / Lijst
4. **Testen:** Titel → Details → Kijklijst / Bekeken
5. **Openen:** Instellingen → Voorkeuren → Collectors → Collectors mode / Merge multiple editions as one title
6. **Configureren:** Collectors Mode staat standaard uit en toont geavanceerde containerfuncties; `merge_editions_as_title` is een afzonderlijke voorkeur die eveneens standaard uitstaat
7. **Testen:** als `merge_editions_as_title` aanstaat, verschijnen meerdere fysieke versies als één bibliotheektegel; open de tegel om elke editie te zien
8. **Vergelijken:** een box-set is een fysiek winkelproduct met een eigen barcode; de films blijven afzonderlijke titels die als directe filmleden zijn gekoppeld
9. **Vergelijken:** een vault is een door de gebruiker gemaakte plank-, opslagdoos- of locatiegroep met directe filmleden en zonder betekenis als winkelproduct
10. **Vergelijken:** een collection is een thematische groep waarvan de geordende items films, box-sets, vaults of andere collections kunnen zijn, zodat nesten wordt ondersteund
11. **Testen:** Collectors Mode uitschakelen verbergt de containerweergave maar verwijdert geen containers of leden
