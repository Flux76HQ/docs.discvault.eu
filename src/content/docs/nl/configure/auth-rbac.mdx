## Procedure

1. **Openen:** Beheer → Beveiliging → Authenticatie inschakelen
2. **Maken:** Beheer → Gebruikers → Uitnodiging voor 48 uur maken
3. **Configureren:** Beheer → Rollen → Basis / Geavanceerd
4. **Testen:** `48 h`
