## Procedure

1. **Controleren:** nieuwe installaties gebruiken DiscVault v26 `:latest` of `:beta`; de bevroren `:legacy`-image is alleen voor een bestaande installatie van de vorige generatie die nog niet klaar is voor migratie
2. **Back-up maken:** bijpassende back-upset
3. **Kiezen:** Alleen de Legacy-route: wijzig na de back-up uitsluitend de imagetag van de bestaande oude implementatie in `:legacy`, haal deze op en maak hem opnieuw met ongewijzigde topologie, controleer de collectie en stop; schakel deze bestaande implementatie op deze pagina niet over naar de v26-topologie
4. **Openen:** [Bijwerken en migreren: Migratie naar DiscVault 26](/nl/update/migration/) 路 DiscVault 26
5. **Bewaren:** houd volumes, omgevingsvariabelen, poorten en het gegevenspad van de vorige generatie ongewijzigd; vervang in de v26 PostgreSQL Compose-topologie nooit een image door `:legacy`
6. **Verifi毛ren:** `DiscVault v26` 路 `PREVIOUS_IMAGE=@sha256:鈥
7. **Kiezen:** `DiscVault v26` 路 `DISCVAULT_IMAGE` 路 `:latest` `:beta`
8. **Uitvoeren:** `DiscVault v26` 路 `--force-recreate`
9. **Vergelijken:** `DiscVault v26` 路 de bedoelde image-digest draait, health slaagt en eigenaarlogin, bibliotheekaantal en 茅茅n mediavoorbeeld werken

### Opdrachten en instellingen

#### DiscVault v26 stabiel

export DISCVAULT_IMAGE=ghcr.io/helmerznl/discvault:latest

#### DiscVault v26 b猫ta

#### Beide kanalen

```bash
set -euo pipefail
test -n "$DISCVAULT_IMAGE"
case "$DISCVAULT_IMAGE" in
  ghcr.io/helmerznl/discvault:latest|ghcr.io/helmerznl/discvault:beta) ;;
  *) printf '%s\n' "$DISCVAULT_IMAGE" >&2; exit 64 ;;
esac
cd /opt/discvault
ENV_FILE=/opt/discvault/.env
ENV_TMP="$(mktemp "$ENV_FILE.update.XXXXXX")"
trap 'rm -f "$ENV_TMP"' EXIT
awk -v image="$DISCVAULT_IMAGE" '
  /^DISCVAULT_IMAGE=/ { print "DISCVAULT_IMAGE=" image; found=1; next }
  { print }
  END { if (!found) exit 42 }
' "$ENV_FILE" > "$ENV_TMP"
chmod 0600 "$ENV_TMP"
docker compose --env-file "$ENV_TMP" -p discvault -f compose.yaml config --quiet
docker compose --env-file "$ENV_TMP" -p discvault -f compose.yaml pull next-api next-worker next-mcp
docker compose --env-file "$ENV_TMP" -p discvault -f compose.yaml up -d --force-recreate
docker compose --env-file "$ENV_TMP" -p discvault -f compose.yaml ps
curl --fail http://localhost:6080/api/next/health
mv "$ENV_TMP" "$ENV_FILE"
trap - EXIT
```
