## Procedure

2. **Configureren:** `RP_ORIGINS` ôñ HTTPS
3. **Configureren:** `http://127.0.0.1:6080`
4. **Testen:** `GET /api/next/health`

### Opdrachten en instellingen

#### Beide kanalen

```text

curl --fail https://discvault.example.com/api/next/health
```
