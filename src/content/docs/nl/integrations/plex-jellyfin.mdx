2. **Testen:** Beheer → Plug-ins → Statuscontrole
3. **Uitvoeren:** Beheer → Plug-ins → Ontdekken zonder opslaan · `dryRun=true`
4. **Vergelijken:** `/api/next/digital-items?limit=200`
