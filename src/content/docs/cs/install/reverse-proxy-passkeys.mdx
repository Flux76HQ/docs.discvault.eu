## Požadavky

- FQDN
- TLS
- `RP_ID`

## Postup

1. **Vyberte:** FQDN · mimo výjimku pro nastavení na localhost vyžadují passkeys stabilní plně kvalifikovaný název domény přes důvěryhodné HTTPS; holá IP adresa ani prosté HTTP nejsou podporovány
2. **Nastavte:** `RP_ID`
3. **Nastavte:** `RP_ORIGINS` · HTTPS
4. **Nastavte:** `http://127.0.0.1:6080`
5. **Otestujte:** `GET /api/next/health`
6. **Zaznamenejte:** zvolte název hostitele před registrací passkeys; pozdější změna vyžaduje novou registraci a localhost zůstává pouze výjimkou pro místní nastavení

### Příkazy a nastavení

#### Oba kanály

```text
RP_ID=discvault.example.com
RP_ORIGINS=https://discvault.example.com
curl --fail https://discvault.example.com/api/next/health
```
