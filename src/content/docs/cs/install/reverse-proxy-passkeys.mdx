## Postup

1. **Nastavte:** `RP_ID`
2. **Nastavte:** `RP_ORIGINS` · HTTPS
3. **Nastavte:** `http://127.0.0.1:6080`
4. **Otestujte:** `GET /api/next/health`

### Příkazy a nastavení

#### Oba kanály

```text

RP_ORIGINS=https://discvault.example.com
curl --fail https://discvault.example.com/api/next/health
```
