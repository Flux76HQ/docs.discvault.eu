## Postup

1. **Zkontrolujte:** `GET /api/next/plugins/registry`
2. **Nastavte:** Správa → Pluginy → Poskytovatel → Nastavit · přihlašovací údaj poskytovatele uložený jako tajný
3. **Otestujte:** Správa → Pluginy → Kontrola stavu · Správa → Pluginy → Najít bez uložení · `dryRun`
4. **Zaznamenejte:** `secret app_settings`

### Příkazy a nastavení

#### Oba kanály

```bash
curl --fail http://localhost:6080/api/next/plugins/registry
curl --fail http://localhost:6080/api/next/metadata/plugins
```
