## Požadavky

- přístup vlastníka
- HTTPS
- druhá metoda obnovení

## Postup

1. **Otevřete:** Správa → Zabezpečení → Zapnout ověřování
2. **Vytvořte:** Správa → Uživatelé → Vytvořit pozvánku na 48 hodin
3. **Nastavte:** Správa → Role → Základní / Pokročilé
4. **Otestujte:** `48 h`

### Příkazy a nastavení

#### Oba kanály

```bash
curl --fail http://localhost:6080/api/next/auth/status
curl --fail http://localhost:6080/api/next/auth/rbac
```
