## Postup

1. **Zkontrolujte:** Docker Engine `≥24` · Docker Compose `v2`
2. **Zkontrolujte:** `≥2 GB` · zapisovatelná trvalá data a volné místo pro zálohu
3. **Nastavte:** DNS · TLS · HTTPS · `RP_ID` · `RP_ORIGINS`
4. **Zaznamenejte:** vybraný kanál vydání
