## Postup

1. **Otevřete:** Knihovna → Hledat
2. **Nastavte:** Knihovna → Filtr → Formát / Žánr / Skupina
3. **Porovnejte:** Knihovna → Zobrazení → Plakáty / Seznam
4. **Otestujte:** Titul → Podrobnosti → Seznam ke zhlédnutí / Zhlédnuto
5. **Nastavte:** Collectors Mode je ve výchozím stavu vypnutý a zpřístupňuje pokročilé funkce kontejnerů; `merge_editions_as_title` je samostatná předvolba, která je rovněž ve výchozím stavu vypnutá
6. **Otestujte:** když je zapnuto `merge_editions_as_title`, více fyzických verzí se zobrazí jako jedna dlaždice knihovny; otevřením dlaždice zobrazíte všechny edice
7. **Porovnejte:** box-set je fyzický maloobchodní produkt s vlastním čárovým kódem; jeho filmy zůstávají samostatnými tituly propojenými jako přímí členové
8. **Porovnejte:** vault je uživatelem vytvořené seskupení police, úložného boxu nebo umístění s přímými filmovými členy a bez významu maloobchodního produktu
9. **Porovnejte:** collection je tematické seskupení, jehož seřazenými položkami mohou být filmy, box-sety, vaulty nebo jiné kolekce, takže podporuje vnoření
10. **Otestujte:** vypnutí Collectors Mode skryje zobrazení kontejnerů, ale neodstraní kontejnery ani jejich členy
