## Procedimiento

1. **Configura:** Administración → Complementos → Proveedor → Configurar · Plex Jellyfin
2. **Prueba:** Administración → Complementos → Comprobar estado
3. **Ejecuta:** Administración → Complementos → Descubrir sin guardar · `dryRun=true`
4. **Compara:** `/api/next/digital-items?limit=200`
