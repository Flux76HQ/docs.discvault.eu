## Procedimiento

1. **Abre:** Biblioteca → Buscar
2. **Configura:** Biblioteca → Filtro → Formato / Género / Grupo
3. **Compara:** Biblioteca → Vista → Pósteres / Lista
4. **Prueba:** Título → Detalles → Lista / Visto
5. **Abre:** Ajustes → Preferencias → Collectors → Collectors mode / Merge multiple editions as one title
6. **Configura:** Collectors Mode está desactivado de forma predeterminada y muestra funciones avanzadas de contenedores; `merge_editions_as_title` es una preferencia independiente que también está desactivada
7. **Prueba:** al activar `merge_editions_as_title`, varias versiones físicas aparecen como una sola ficha de biblioteca; abre la ficha para ver todas las ediciones
8. **Compara:** un box-set es un producto físico de venta con su propio código de barras; sus películas siguen siendo títulos individuales enlazados como miembros directos
9. **Compara:** un vault es una agrupación creada por el usuario para una estantería, caja o ubicación, con películas directas y sin significado de producto comercial
10. **Compara:** una collection es una agrupación temática cuyos elementos ordenados pueden ser películas, box-sets, vaults u otras collections, por lo que admite anidación
11. **Prueba:** desactivar Collectors Mode oculta la presentación de contenedores, pero no elimina los contenedores ni sus miembros
