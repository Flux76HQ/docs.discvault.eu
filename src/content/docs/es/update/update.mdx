## Procedimiento

1. **Inspecciona:** las instalaciones nuevas usan DiscVault v26 `:latest` o `:beta`; la imagen congelada `:legacy` es solo para una instalación existente de la generación anterior que aún no esté lista para migrar
2. **Crea una copia:** juego de copias coincidente
3. **Elige:** Solo rama Legacy: después de la copia, cambia únicamente la etiqueta de imagen del despliegue antiguo existente a `:legacy`, descárgalo y recréalo con la topología intacta, verifica la colección y detente; no cambies ese despliegue existente a la topología v26 en esta página
4. **Abre:** [Actualización y migración: Migración a DiscVault 26](/es/update/migration/) · DiscVault 26
5. **Conserva:** mantén sin cambios los volúmenes, variables de entorno, puertos y ruta de datos de la generación anterior; nunca sustituyas una imagen de la topología Compose v26 con PostgreSQL por `:legacy`
6. **Verifica:** `DiscVault v26` · `PREVIOUS_IMAGE=@sha256:…`
7. **Elige:** `DiscVault v26` · `DISCVAULT_IMAGE` · `:latest` `:beta`
8. **Ejecuta:** `DiscVault v26` · `--force-recreate`
9. **Compara:** `DiscVault v26` · se ejecuta el digest previsto, la salud es correcta y pasan el acceso del propietario, el recuento y una muestra multimedia

### Comandos y ajustes

#### DiscVault v26 estable

```bash
export DISCVAULT_IMAGE=ghcr.io/helmerznl/discvault:latest
```

#### DiscVault v26 beta

```bash
export DISCVAULT_IMAGE=ghcr.io/helmerznl/discvault:beta
```

#### Ambos canales

```bash
set -euo pipefail
test -n "$DISCVAULT_IMAGE"
case "$DISCVAULT_IMAGE" in
  ghcr.io/helmerznl/discvault:latest|ghcr.io/helmerznl/discvault:beta) ;;
  *) printf '%s\n' "$DISCVAULT_IMAGE" >&2; exit 64 ;;
esac
cd /opt/discvault
ENV_FILE=/opt/discvault/.env
ENV_TMP="$(mktemp "$ENV_FILE.update.XXXXXX")"
trap 'rm -f "$ENV_TMP"' EXIT
awk -v image="$DISCVAULT_IMAGE" '
  /^DISCVAULT_IMAGE=/ { print "DISCVAULT_IMAGE=" image; found=1; next }
  { print }
  END { if (!found) exit 42 }
' "$ENV_FILE" > "$ENV_TMP"
chmod 0600 "$ENV_TMP"
docker compose --env-file "$ENV_TMP" -p discvault -f compose.yaml config --quiet
docker compose --env-file "$ENV_TMP" -p discvault -f compose.yaml pull next-api next-worker next-mcp
docker compose --env-file "$ENV_TMP" -p discvault -f compose.yaml up -d --force-recreate
docker compose --env-file "$ENV_TMP" -p discvault -f compose.yaml ps
curl --fail http://localhost:6080/api/next/health
mv "$ENV_TMP" "$ENV_FILE"
trap - EXIT
```
