## Objetivo y resultado

Completa el flujo específico de la página para **Configuración: Autenticación y RBAC**. Este procedimiento utiliza `/api/next/auth/status` · `/api/next/auth/rbac` · WebAuthn · `48 h`.

## Aplicable a

Esta página se aplica solo a **Configuración: Autenticación y RBAC** (web). Sus límites de canal y datos son: DiscVault · `ghcr.io/helmerznl/discvault:latest`; WebAuthn; DiscVault · `ghcr.io/helmerznl/discvault:beta`; WebAuthn. Puntos de control de esta página: `/api/next/auth/status` · `/api/next/auth/rbac` · WebAuthn · `48 h`.

### DiscVault v26 estable

- DiscVault · `ghcr.io/helmerznl/discvault:latest`
- WebAuthn

### DiscVault v26 beta

- DiscVault · `ghcr.io/helmerznl/discvault:beta`

## Requisitos previos

- acceso de propietario
- HTTPS
- segundo método de recuperación

## Procedimiento

1. **Abre:** Administración → Seguridad → Activar autenticación
2. **Crea:** Administración → Usuarios → Crear invitación de 48 horas
3. **Configura:** Administración → Roles → Básico / Avanzado
4. **Prueba:** `48 h`

### Comandos y ajustes

#### Ambos canales

```bash
curl --fail http://localhost:6080/api/next/auth/status
curl --fail http://localhost:6080/api/next/auth/rbac
```

## Resultado esperado

Acepta el resultado de **Configuración: Autenticación y RBAC** solo cuando se cumplan todos estos resultados: el acceso del propietario con passkey funciona, el registro por invitación coincide con el ajuste y el usuario de prueba solo tiene sus permisos. Puntos de control de esta página: `/api/next/auth/status` · `/api/next/auth/rbac` · WebAuthn · `48 h`.

- el acceso del propietario con passkey funciona, el registro por invitación coincide con el ajuste y el usuario de prueba solo tiene sus permisos

## Seguridad y reversión

:::caution
El límite de recuperación segura de **Configuración: Autenticación y RBAC** exige estas acciones antes de iniciar o continuar: mantén la sesión del propietario y un segundo método; no elimines nunca la última passkey del propietario. Puntos de control de esta página: `/api/next/auth/status` · `/api/next/auth/rbac` · WebAuthn · `48 h`.

- mantén la sesión del propietario y un segundo método; no elimines nunca la última passkey del propietario
:::
