## Requisitos previos

- sesión de propietario o administrador
- credencial del proveedor guardada como secreto

## Procedimiento

1. **Inspecciona:** `GET /api/next/plugins/registry`
2. **Configura:** Administración → Complementos → Proveedor → Configurar · credencial del proveedor guardada como secreto
3. **Prueba:** Administración → Complementos → Comprobar estado · Administración → Complementos → Descubrir sin guardar · `dryRun`
4. **Registra:** `secret app_settings`

### Comandos y ajustes

#### Ambos canales

```bash
curl --fail http://localhost:6080/api/next/plugins/registry
curl --fail http://localhost:6080/api/next/metadata/plugins
```
