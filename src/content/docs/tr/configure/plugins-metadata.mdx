## Prosedür

1. **İnceleyin:** `GET /api/next/plugins/registry`
2. **Yapılandırın:** Yönetim → Eklentiler → Sağlayıcı → Yapılandır · gizli olarak saklanan sağlayıcı kimlik bilgisi
3. **Test edin:** Yönetim → Eklentiler → Durum denetimi · Yönetim → Eklentiler → Kaydetmeden keşfet · `dryRun`
4. **Kaydedin:** `secret app_settings`
