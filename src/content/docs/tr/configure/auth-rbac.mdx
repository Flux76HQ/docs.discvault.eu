## Prosedür

1. **Açın:** Yönetim → Güvenlik → Kimlik doğrulamayı etkinleştir
2. **Oluşturun:** Yönetim → Kullanıcılar → 48 saatlik davet oluştur
3. **Yapılandırın:** Yönetim → Roller → Temel / Gelişmiş
4. **Test edin:** `48 h`
