## Prosedür

1. **İnceleyin:** yeni kurulumlar DiscVault v26 `:latest` veya `:beta` kullanır; dondurulmuş `:legacy` imajı yalnızca henüz geçişe hazır olmayan mevcut önceki nesil kurulum içindir
2. **Yedekleyin:** eşleşen yedek kümesi
3. **Seçin:** Yalnızca Legacy dalı: yedekten sonra mevcut eski dağıtımın yalnızca imaj etiketini `:legacy` olarak değiştirin, topolojiyi değiştirmeden çekip yeniden oluşturun, koleksiyonu doğrulayıp durun; bu sayfada mevcut o dağıtımı v26 topolojisine geçirmeyin
4. **Açın:** [Güncelleme ve geçiş: DiscVault 26 geçişi](/tr/update/migration/) · DiscVault 26
5. **Koruyun:** önceki neslin birimlerini, ortam değişkenlerini, bağlantı noktalarını ve veri yolunu değiştirmeyin; v26 PostgreSQL Compose topolojisindeki imajı asla `:legacy` ile değiştirmeyin
6. **Doğrulayın:** `DiscVault v26` · `PREVIOUS_IMAGE=@sha256:…`
7. **Seçin:** `DiscVault v26` · `DISCVAULT_IMAGE` · `:latest` `:beta`
8. **Çalıştırın:** `DiscVault v26` · `--force-recreate`
9. **Karşılaştırın:** `DiscVault v26` · amaçlanan imaj özeti çalışır, sağlık başarılıdır ve sahip girişi, kitaplık sayısı ile bir medya örneği geçer

### Komutlar ve ayarlar

#### DiscVault v26 kararlı

```bash
export DISCVAULT_IMAGE=ghcr.io/helmerznl/discvault:latest
```

#### DiscVault v26 beta

```bash
export DISCVAULT_IMAGE=ghcr.io/helmerznl/discvault:beta
```

#### Her iki kanal

```bash
set -euo pipefail
test -n "$DISCVAULT_IMAGE"
case "$DISCVAULT_IMAGE" in
  ghcr.io/helmerznl/discvault:latest|ghcr.io/helmerznl/discvault:beta) ;;
  *) printf '%s\n' "$DISCVAULT_IMAGE" >&2; exit 64 ;;
esac
cd /opt/discvault
ENV_FILE=/opt/discvault/.env
ENV_TMP="$(mktemp "$ENV_FILE.update.XXXXXX")"
trap 'rm -f "$ENV_TMP"' EXIT
awk -v image="$DISCVAULT_IMAGE" '
  /^DISCVAULT_IMAGE=/ { print "DISCVAULT_IMAGE=" image; found=1; next }
  { print }
  END { if (!found) exit 42 }
' "$ENV_FILE" > "$ENV_TMP"
chmod 0600 "$ENV_TMP"
docker compose --env-file "$ENV_TMP" -p discvault -f compose.yaml config --quiet
docker compose --env-file "$ENV_TMP" -p discvault -f compose.yaml pull next-api next-worker next-mcp
docker compose --env-file "$ENV_TMP" -p discvault -f compose.yaml up -d --force-recreate
docker compose --env-file "$ENV_TMP" -p discvault -f compose.yaml ps
curl --fail http://localhost:6080/api/next/health
mv "$ENV_TMP" "$ENV_FILE"
trap - EXIT
```
