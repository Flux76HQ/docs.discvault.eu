## Prosedür

1. **Açın:** Kitaplık → Ara
2. **Yapılandırın:** Kitaplık → Filtre → Biçim / Tür / Grup
3. **Karşılaştırın:** Kitaplık → Görünüm → Posterler / Liste
4. **Test edin:** Başlık → Ayrıntılar → İzleme listesi / İzlendi
5. **Açın:** Ayarlar → Tercihler → Collectors → Collectors mode / Merge multiple editions as one title
6. **Yapılandırın:** Collectors Mode varsayılan olarak kapalıdır ve gelişmiş kapsayıcı özelliklerini gösterir; `merge_editions_as_title` yine varsayılan olarak kapalı ayrı bir tercihtir
7. **Test edin:** `merge_editions_as_title` etkinleştirildiğinde birden çok fiziksel sürüm tek kitaplık kutucuğu olarak görünür; tüm sürümleri görmek için kutucuğu açın
8. **Karşılaştırın:** box-set kendi barkodu olan fiziksel perakende ürünüdür; filmleri doğrudan film üyeleri olarak bağlı ayrı başlıklar halinde kalır
9. **Karşılaştırın:** vault kullanıcı tarafından oluşturulan raf, saklama kutusu veya konum grubudur; doğrudan film üyeleri vardır ve perakende ürün anlamı taşımaz
10. **Karşılaştırın:** collection sıralı öğeleri film, box-set, vault veya başka collection olabilen tematik gruptur; iç içe yerleştirme desteklenir
11. **Test edin:** Collectors Mode kapatıldığında kapsayıcı sunumu gizlenir ancak kapsayıcılar veya üyeleri silinmez
