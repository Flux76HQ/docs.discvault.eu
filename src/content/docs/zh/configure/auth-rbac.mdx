## 前提条件

- 所有者权限
- HTTPS
- 第二种恢复方式

## 操作步骤

1. **打开:** 管理 → 安全 → 启用身份验证
2. **创建:** 管理 → 用户 → 创建 48 小时邀请
3. **配置:** 管理 → 角色 → 基础 / 高级
4. **测试:** `48 h`

### 命令和设置

#### 两个通道

```bash
curl --fail http://localhost:6080/api/next/auth/status
curl --fail http://localhost:6080/api/next/auth/rbac
```
