- 具有所需权限的已登录账号
- `watchlist`

## 操作步骤

1. **打开:** 媒体库 → 搜索
2. **配置:** 媒体库 → 筛选 → 格式 / 类型 / 分组
3. **比较:** 媒体库 → 视图 → 海报 / 列表
4. **测试:** 标题 → 详情 → 想看 / 已看
5. **打开:** 设置 → 偏好设置 → Collectors → Collectors mode / Merge multiple editions as one title
6. **配置:** Collectors Mode 默认关闭并显示高级容器功能；`merge_editions_as_title` 是另一项同样默认关闭的独立偏好设置
7. **测试:** 启用 `merge_editions_as_title` 后，多个实体版本显示为一个媒体库磁贴；打开磁贴可查看每个版本
8. **比较:** box-set 是带有自身条码的实体零售产品；其中的影片仍是作为直接影片成员关联的独立标题
9. **比较:** vault 是用户创建的货架、储物箱或位置分组，包含直接影片成员，不代表零售产品
10. **比较:** collection 是主题分组，其有序项目可以是影片、box-set、vault 或其他 collection，因此支持嵌套
11. **测试:** 关闭 Collectors Mode 会隐藏容器展示，但不会删除容器或其成员
