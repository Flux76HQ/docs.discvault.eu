Set up the Android offline-first foundation from a verified test distribution.

## Applies to

Beta foundation/MVP in source. Store availability is not assumed. Local Room storage, library, add flow, and settings must be verified in the installed build.

## Prerequisites

- Android `8.0+` / API `26+`
- build from a verified distribution source
- camera permission

## Procedure

1. **Open:** Launch → Onboarding → Create local library · Android
2. **Create:** Library → Add → Scan barcode / Enter manually · one test title
3. **Test:** app restart · Room persistence
4. **Test:** airplane mode · the test title remains after app restart and an offline launch



## Expected result
