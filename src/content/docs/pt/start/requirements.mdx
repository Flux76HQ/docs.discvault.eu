## Pré-requisitos

- Docker Engine `24+`
- armazenamento persistente gravável e espaço livre

## Procedimento

1. **Inspecionar:** Docker Engine `≥24` · Docker Compose `v2`
2. **Inspecionar:** `≥2 GB` · armazenamento persistente gravável e espaço livre
3. **Configurar:** DNS · TLS · HTTPS · `RP_ID` · `RP_ORIGINS`
4. **Registar:** canal de lançamento selecionado
