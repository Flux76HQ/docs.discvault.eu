## Pré-requisitos

- conta autenticada com a permissão necessária
- `watchlist`

## Procedimento

1. **Abrir:** Biblioteca → Pesquisar
2. **Configurar:** Biblioteca → Filtro → Formato / Género / Grupo
3. **Comparar:** Biblioteca → Vista → Cartazes / Lista
4. **Testar:** Título → Detalhes → Lista / Visto
5. **Abrir:** Definições → Preferências → Collectors → Collectors mode / Merge multiple editions as one title
6. **Configurar:** Collectors Mode está desativado por predefinição e apresenta funções avançadas de contentores; `merge_editions_as_title` é uma preferência separada, também desativada por predefinição
7. **Testar:** quando `merge_editions_as_title` está ativo, várias versões físicas aparecem como um único mosaico; abra-o para ver todas as edições
8. **Comparar:** um box-set é um produto físico de venda com código de barras próprio; os filmes continuam títulos individuais ligados como membros diretos
9. **Comparar:** um vault é um agrupamento criado pelo utilizador para prateleira, caixa ou localização, com filmes diretos e sem significado de produto comercial
10. **Comparar:** uma collection é um agrupamento temático cujos itens ordenados podem ser filmes, box-sets, vaults ou outras collections, permitindo aninhamento
11. **Testar:** desativar Collectors Mode oculta a apresentação dos contentores, mas não elimina contentores nem membros

:::caution
