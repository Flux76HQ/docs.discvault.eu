## Procedimento

1. **Inspecionar:** as novas instalações usam DiscVault v26 `:latest` ou `:beta`; a imagem congelada `:legacy` destina-se apenas a uma instalação existente da geração anterior que ainda não esteja pronta para migrar
2. **Criar cópia:** conjunto de cópias correspondente
3. **Escolher:** Apenas o ramo Legacy: após a cópia, altere somente a tag da imagem da implementação antiga existente para `:legacy`, transfira e recrie-a sem mudar a topologia, confirme a coleção e pare; não execute os comandos v26 abaixo
4. **Abrir:** [Atualização e migração: Migração para DiscVault 26](/pt/update/migration/) · DiscVault 26
5. **Preservar:** mantenha inalterados os volumes, variáveis de ambiente, portas e caminho de dados da geração anterior; nunca substitua uma imagem na topologia Compose v26 PostgreSQL por `:legacy`
6. **Verificar:** `DiscVault v26` · `PREVIOUS_IMAGE=@sha256:…`
7. **Escolher:** `DiscVault v26` · `DISCVAULT_IMAGE` · `:latest` `:beta`
8. **Executar:** `DiscVault v26` · `--force-recreate`
9. **Comparar:** `DiscVault v26` · o digest pretendido está em execução, a saúde é bem-sucedida e o login, a contagem e uma amostra passam

### Comandos e definições

#### DiscVault v26 estável

```bash
export DISCVAULT_IMAGE=ghcr.io/helmerznl/discvault:latest
```

#### DiscVault v26 beta

```bash
export DISCVAULT_IMAGE=ghcr.io/helmerznl/discvault:beta
```

#### Ambos os canais

```bash
set -euo pipefail
test -n "$DISCVAULT_IMAGE"
case "$DISCVAULT_IMAGE" in
  ghcr.io/helmerznl/discvault:latest|ghcr.io/helmerznl/discvault:beta) ;;
  *) printf '%s\n' "$DISCVAULT_IMAGE" >&2; exit 64 ;;
esac
cd /opt/discvault
ENV_FILE=/opt/discvault/.env
ENV_TMP="$(mktemp "$ENV_FILE.update.XXXXXX")"
trap 'rm -f "$ENV_TMP"' EXIT
awk -v image="$DISCVAULT_IMAGE" '
  /^DISCVAULT_IMAGE=/ { print "DISCVAULT_IMAGE=" image; found=1; next }
  { print }
  END { if (!found) exit 42 }
' "$ENV_FILE" > "$ENV_TMP"
chmod 0600 "$ENV_TMP"
docker compose --env-file "$ENV_TMP" -p discvault -f compose.yaml config --quiet
docker compose --env-file "$ENV_TMP" -p discvault -f compose.yaml pull next-api next-worker next-mcp
docker compose --env-file "$ENV_TMP" -p discvault -f compose.yaml up -d --force-recreate
docker compose --env-file "$ENV_TMP" -p discvault -f compose.yaml ps
curl --fail http://localhost:6080/api/next/health
mv "$ENV_TMP" "$ENV_FILE"
trap - EXIT
```
