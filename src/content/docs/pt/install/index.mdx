## Procedimento

1. **Escolher:** [Instalação: Docker Compose](/pt/install/docker-compose/)
2. **Escolher:** [Instalação: Docker run](/pt/install/docker-run/)
3. **Escolher:** [Instalação: Unraid](/pt/install/unraid/) · Unraid
4. **Escolher:** as novas instalações usam DiscVault v26 `:latest` ou `:beta`; a imagem congelada `:legacy` destina-se apenas a uma instalação existente da geração anterior que ainda não esteja pronta para migrar
5. **Preservar:** mantenha inalterados os volumes, variáveis de ambiente, portas e caminho de dados da geração anterior; nunca substitua uma imagem na topologia Compose v26 PostgreSQL por `:legacy`
6. **Abrir:** [Atualização e migração: Migração para DiscVault 26](/pt/update/migration/) · → DiscVault 26
7. **Registar:** `DISCVAULT_IMAGE` · conjunto de cópias correspondente
