## Pré-requisitos

- DNS
- TLS
- `6080`

## Procedimento

1. **Configurar:** `RP_ID`
2. **Configurar:** `RP_ORIGINS` · HTTPS
3. **Configurar:** `http://127.0.0.1:6080`
4. **Testar:** `GET /api/next/health`

### Comandos e definições

#### Ambos os canais

```text
RP_ID=discvault.example.com
RP_ORIGINS=https://discvault.example.com
curl --fail https://discvault.example.com/api/next/health
```
