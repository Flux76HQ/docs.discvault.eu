## Pré-requisitos

- credencial do fornecedor guardada como segredo
- Jellyfin
- sessão de proprietário ou administrador

## Procedimento

1. **Configurar:** Administração → Plugins → Fornecedor → Configurar · Plex Jellyfin
2. **Testar:** Administração → Plugins → Verificação de estado
3. **Executar:** Administração → Plugins → Descobrir sem guardar · `dryRun=true`
4. **Comparar:** `/api/next/digital-items?limit=200`

### Comandos e definições

#### Ambos os canais

```bash
curl --fail http://localhost:6080/api/next/digital-sources
curl --fail 'http://localhost:6080/api/next/digital-items?limit=200'
```

:::caution
O limite de recuperação segura de **Integrações: Plex e Jellyfin** exige estas ações antes de iniciar ou continuar: desative o fornecedor antes de remover, copie `/data/plugins` e revogue credenciais expostas; desative a integração, revogue credenciais de teste ou expostas, crie uma nova se necessário e nunca reutilize a antiga.
