## Prérequis

- compte connecté avec l’autorisation requise
- `watchlist`

## Procédure

1. **Ouvrir:** Bibliothèque → Rechercher
2. **Configurer:** Bibliothèque → Filtrer → Format / Genre / Groupe
3. **Comparer:** Bibliothèque → Affichage → Affiches / Liste
4. **Tester:** Titre → Détails → Liste à voir / Vu
5. **Ouvrir:** Réglages → Préférences → Collectors → Collectors mode / Merge multiple editions as one title
6. **Configurer:** Collectors Mode est désactivé par défaut et expose les fonctions avancées de conteneurs ; `merge_editions_as_title` est un réglage distinct, lui aussi désactivé par défaut
7. **Tester:** quand `merge_editions_as_title` est activé, plusieurs versions physiques apparaissent sur une seule tuile ; ouvrez-la pour voir chaque édition
8. **Comparer:** un box-set est un produit physique vendu avec son propre code-barres ; ses films restent des titres individuels liés comme membres directs
9. **Comparer:** un vault est un regroupement créé par l’utilisateur pour une étagère, une boîte ou un emplacement, avec des films directs et sans signification de produit commercial
10. **Comparer:** une collection est un regroupement thématique dont les éléments ordonnés peuvent être des films, box-sets, vaults ou autres collections, ce qui permet l’imbrication
11. **Tester:** désactiver Collectors Mode masque la présentation des conteneurs mais ne supprime ni les conteneurs ni leurs membres
