Suivez le déroulement propre à cette page pour **Intégrations: Plex et Jellyfin**. Cette procédure utilise Plex · Jellyfin · `dryRun` · `/api/next/digital-items`.

## Prérequis

- identifiant fournisseur stocké comme secret
- Jellyfin
- session propriétaire ou administrateur

## Procédure

1. **Configurer:** Administration → Plugins → Fournisseur → Configurer · Plex Jellyfin
2. **Tester:** Administration → Plugins → Contrôle de fonctionnement
3. **Exécuter:** Administration → Plugins → Découvrir sans enregistrer · `dryRun=true`
4. **Comparer:** `/api/next/digital-items?limit=200`

### Commandes et réglages

#### Les deux canaux

```bash
curl --fail http://localhost:6080/api/next/digital-sources
curl --fail 'http://localhost:6080/api/next/digital-items?limit=200'
```
