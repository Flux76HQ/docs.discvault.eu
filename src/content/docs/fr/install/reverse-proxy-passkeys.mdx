## Prérequis

- DNS
- TLS
- `6080`

## Procédure

1. **Configurer:** `RP_ID`
2. **Configurer:** `RP_ORIGINS` · HTTPS
3. **Configurer:** `http://127.0.0.1:6080`
4. **Tester:** `GET /api/next/health`

### Commandes et réglages

#### Les deux canaux

```text
RP_ID=discvault.example.com
RP_ORIGINS=https://discvault.example.com
curl --fail https://discvault.example.com/api/next/health
```
