## Objectif et résultat

Suivez le déroulement propre à cette page pour **Démarrer: Vue d’ensemble**. Cette procédure utilise Docker Engine `24+` · Docker Compose `v2` · PostgreSQL `17` · HTTPS.

## S’applique à

Cette page s’applique uniquement à **Démarrer: Vue d’ensemble** (toutes les plateformes). Les limites de canal et de données sont : DiscVault · `ghcr.io/helmerznl/discvault:latest`; Docker Compose `v2` · PostgreSQL `17` · `/data`; DiscVault · `ghcr.io/helmerznl/discvault:beta`; Docker Compose `v2` · PostgreSQL `17` · `/data`. ## Prérequis

- Docker Engine `24+`
- stockage persistant inscriptible et espace de sauvegarde libre
- DNS TLS

## Procédure

1. **Inspecter:** Docker Engine `≥24` · Docker Compose `v2`
2. **Inspecter:** `≥2 GB` · stockage persistant inscriptible et espace de sauvegarde libre
3. **Configurer:** DNS · TLS · HTTPS · `RP_ID` · `RP_ORIGINS`
4. **Consigner:** canal de publication choisi

### Commandes et réglages

#### Les deux canaux

```bash
docker version --format '{{.Server.Version}}'
docker compose version
curl --version
```
