- HTTPS

## Vorgehen

1. **Öffnen:** Administration → Sicherheit → Authentifizierung aktivieren
2. **Erstellen:** Administration → Benutzer → 48-Stunden-Einladung erstellen
3. **Konfigurieren:** Administration → Rollen → Einfach / Erweitert
4. **Testen:** `48 h`

### Befehle und Einstellungen

curl --fail http://localhost:6080/api/next/auth/status
