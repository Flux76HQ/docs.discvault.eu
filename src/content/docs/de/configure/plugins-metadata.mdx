## Voraussetzungen

- Owner- oder Administratorsitzung
- als Secret gespeicherte Anbieter-Zugangsdaten

## Vorgehen

1. **Prüfen:** `GET /api/next/plugins/registry`
2. **Konfigurieren:** Administration → Plugins → Anbieter → Konfigurieren · als Secret gespeicherte Anbieter-Zugangsdaten
3. **Testen:** Administration → Plugins → Zustandsprüfung · Administration → Plugins → Erkennen ohne Speichern · `dryRun`
4. **Festhalten:** `secret app_settings`

### Befehle und Einstellungen

#### Beide Kanäle

```bash
curl --fail http://localhost:6080/api/next/plugins/registry
curl --fail http://localhost:6080/api/next/metadata/plugins
```
