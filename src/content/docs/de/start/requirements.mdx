## Voraussetzungen

- Docker Engine `24+`
- beschreibbarer persistenter Speicher und freier Sicherungsplatz
- DNS TLS

## Vorgehen

1. **Prüfen:** Docker Engine `≥24` · Docker Compose `v2`
2. **Prüfen:** `≥2 GB` · beschreibbarer persistenter Speicher und freier Sicherungsplatz
3. **Konfigurieren:** DNS · TLS · HTTPS · `RP_ID` · `RP_ORIGINS`
4. **Festhalten:** gewählter Release-Kanal
