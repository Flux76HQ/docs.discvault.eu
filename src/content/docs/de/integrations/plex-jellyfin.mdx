## Vorgehen

1. **Konfigurieren:** Administration → Plugins → Anbieter → Konfigurieren · Plex Jellyfin
2. **Testen:** Administration → Plugins → Zustandsprüfung
3. **Ausführen:** Administration → Plugins → Erkennen ohne Speichern · `dryRun=true`
4. **Vergleichen:** `/api/next/digital-items?limit=200`
