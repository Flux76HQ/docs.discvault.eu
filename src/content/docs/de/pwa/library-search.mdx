## Vorgehen

1. **Öffnen:** Bibliothek → Suchen
2. **Konfigurieren:** Bibliothek → Filter → Format / Genre / Gruppe
3. **Vergleichen:** Bibliothek → Ansicht → Poster / Liste
4. **Testen:** Titel → Details → Merkliste / Gesehen
5. **Konfigurieren:** Collectors Mode ist standardmäßig aus und aktiviert erweiterte Containerfunktionen; `merge_editions_as_title` ist eine separate, ebenfalls standardmäßig deaktivierte Einstellung
6. **Testen:** wenn `merge_editions_as_title` aktiviert ist, erscheinen mehrere physische Versionen als eine Bibliothekskachel; die Kachel zeigt alle Editionen
7. **Vergleichen:** ein Box-Set ist ein physisches Handelsprodukt mit eigenem Barcode; seine Filme bleiben einzelne Titel und sind als direkte Filmmitglieder verknüpft
8. **Vergleichen:** ein Vault ist eine benutzerdefinierte Regal-, Lagerbox- oder Ortsgruppe mit direkten Filmmitgliedern und ohne Bedeutung als Handelsprodukt
9. **Vergleichen:** eine Collection ist eine thematische Gruppe, deren geordnete Elemente Filme, Box-Sets, Vaults oder andere Collections sein können; Verschachtelung wird unterstützt
10. **Testen:** das Ausschalten von Collectors Mode blendet die Containerdarstellung aus, löscht aber weder Container noch deren Mitglieder

:::caution
Die sichere Wiederherstellungsgrenze für **PWA: Bibliothek und Suche** verlangt vor jedem Start oder Fortsetzen diese Maßnahmen: das Ausschalten von Collectors Mode blendet die Containerdarstellung aus, löscht aber weder Container noch deren Mitglieder; falsche Bearbeitung vor dem Speichern abbrechen, gespeichertes Feld in der UI zurücksetzen und bei Massenschäden passende Serversicherung verwenden.
