## Мета й результат

Виконайте спеціальну процедуру сторінки для **Налаштування: Автентифікація та RBAC**. У цій процедурі використовуються `/api/next/auth/status` · `/api/next/auth/rbac` · WebAuthn · `48 h`.

## Процедура

1. **Відкрийте:** Адміністрування → Безпека → Увімкнути автентифікацію
2. **Створіть:** Адміністрування → Користувачі → Створити запрошення на 48 годин
3. **Налаштуйте:** Адміністрування → Ролі → Базовий / Розширений
4. **Перевірте:** `48 h`

### Команди й налаштування

#### Обидва канали

```bash
curl --fail http://localhost:6080/api/next/auth/status
curl --fail http://localhost:6080/api/next/auth/rbac
```
