## Процедура

1. **Перевірте:** `GET /api/next/plugins/registry`
2. **Налаштуйте:** Адміністрування → Плагіни → Постачальник → Налаштувати · облікові дані постачальника, збережені як секрет
3. **Перевірте:** Адміністрування → Плагіни → Перевірка стану · Адміністрування → Плагіни → Виявити без збереження · `dryRun`
4. **Запишіть:** `secret app_settings`

### Команди й налаштування

#### Обидва канали

```bash
curl --fail http://localhost:6080/api/next/plugins/registry
curl --fail http://localhost:6080/api/next/metadata/plugins
```
