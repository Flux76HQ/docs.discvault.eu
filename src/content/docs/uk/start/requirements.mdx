## Процедура

1. **Перевірте:** Docker Engine `≥24` · Docker Compose `v2`
2. **Перевірте:** `≥2 GB` · доступне для запису постійне сховище й вільне місце
3. **Налаштуйте:** DNS · TLS · HTTPS · `RP_ID` · `RP_ORIGINS`
4. **Запишіть:** вибраний канал випуску
