## Процедура

1. **Відкрийте:** Бібліотека → Пошук
2. **Налаштуйте:** Бібліотека → Фільтр → Формат / Жанр / Група
3. **Порівняйте:** Бібліотека → Вигляд → Постери / Список
4. **Перевірте:** Назва → Відомості → Список перегляду / Переглянуто
5. **Відкрийте:** Налаштування → Уподобання → Collectors → Collectors mode / Merge multiple editions as one title
6. **Налаштуйте:** Collectors Mode типово вимкнений і відкриває розширені функції контейнерів; `merge_editions_as_title` — окреме налаштування, яке також типово вимкнене
7. **Перевірте:** коли `merge_editions_as_title` увімкнено, кілька фізичних версій показуються однією плиткою бібліотеки; відкрийте її, щоб побачити всі видання
8. **Порівняйте:** box-set — це фізичний товар із власним штрихкодом; його фільми лишаються окремими назвами, пов’язаними як прямі елементи
9. **Порівняйте:** vault — це створене користувачем групування полиці, коробки чи місця з прямими фільмами, без значення роздрібного товару
10. **Порівняйте:** collection — це тематичне групування, впорядкованими елементами якого можуть бути фільми, box-sets, vaults або інші collections, тому підтримується вкладення
11. **Перевірте:** вимкнення Collectors Mode приховує представлення контейнерів, але не видаляє контейнери чи їхні елементи
