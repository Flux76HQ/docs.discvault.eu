1. **Перевірте:** Бібліотека → Пошук · Бібліотека → Додати → Сканувати штрихкод / Ввести вручну · режим польоту
2. **Створіть:** локальна бібліотека з пріоритетом офлайн · тестова назва зберігається після перезапуску й запуску офлайн
3. **Налаштуйте:** Налаштування → Блокування застосунку · Налаштування → Підключення до сервера
4. **Порівняйте:** необов’язкове підключення до сервера, перевірене на одному елементі · один тестовий елемент збігається локально й на сервері або синхронізація лишається вимкненою
