## Eljárás

1. **Ellenőrizze:** az új telepítések a DiscVault v26 `:latest` vagy `:beta` képet használják; a befagyasztott `:legacy` kép csak olyan meglévő előző generációs telepítéshez való, amely még nem áll készen a migrációra
2. **Készítsen mentést:** illeszkedő mentéskészlet
3. **Válassza ki:** Csak Legacy ág: a mentés után csak a meglévő régi telepítés képcímkéjét állítsa `:legacy` értékre, változatlan topológiával töltse le és hozza létre újra, ellenőrizze a gyűjteményt, majd álljon meg; ezen az oldalon ne állítsa át a meglévő telepítést a v26 topológiára
4. **Nyissa meg:** [Frissítés és migrálás: Átállás DiscVault 26-ra](/hu/update/migration/) · DiscVault 26
5. **Őrizze meg:** hagyja változatlanul az előző generáció köteteit, környezeti változóit, portjait és adatútvonalát; a v26 PostgreSQL Compose topológiában soha ne használja a `:legacy` képet
6. **Ellenőrizze:** `DiscVault v26` · `PREVIOUS_IMAGE=@sha256:…`
7. **Válassza ki:** `DiscVault v26` · `DISCVAULT_IMAGE` · `:latest` `:beta`
8. **Futtassa:** `DiscVault v26` · `--force-recreate`
9. **Hasonlítsa össze:** `DiscVault v26` · a kívánt image-digest fut, az állapot egészséges, és a tulajdonosi bejelentkezés, darabszám és médiaminta sikeres

### Parancsok és beállítások

#### DiscVault v26 – stabil

```bash
export DISCVAULT_IMAGE=ghcr.io/helmerznl/discvault:latest
```

#### DiscVault v26 – béta

```bash
export DISCVAULT_IMAGE=ghcr.io/helmerznl/discvault:beta
```

#### Mindkét csatorna

```bash
set -euo pipefail
test -n "$DISCVAULT_IMAGE"
case "$DISCVAULT_IMAGE" in
  ghcr.io/helmerznl/discvault:latest|ghcr.io/helmerznl/discvault:beta) ;;
  *) printf '%s\n' "$DISCVAULT_IMAGE" >&2; exit 64 ;;
esac
cd /opt/discvault
ENV_FILE=/opt/discvault/.env
ENV_TMP="$(mktemp "$ENV_FILE.update.XXXXXX")"
trap 'rm -f "$ENV_TMP"' EXIT
awk -v image="$DISCVAULT_IMAGE" '
  /^DISCVAULT_IMAGE=/ { print "DISCVAULT_IMAGE=" image; found=1; next }
  { print }
  END { if (!found) exit 42 }
' "$ENV_FILE" > "$ENV_TMP"
chmod 0600 "$ENV_TMP"
docker compose --env-file "$ENV_TMP" -p discvault -f compose.yaml config --quiet
docker compose --env-file "$ENV_TMP" -p discvault -f compose.yaml pull next-api next-worker next-mcp
docker compose --env-file "$ENV_TMP" -p discvault -f compose.yaml up -d --force-recreate
docker compose --env-file "$ENV_TMP" -p discvault -f compose.yaml ps
curl --fail http://localhost:6080/api/next/health
mv "$ENV_TMP" "$ENV_FILE"
trap - EXIT
```
