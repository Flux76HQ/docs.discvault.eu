## Eljárás

1. **Válassza ki:** FQDN · a localhost kivételén kívül a passkey stabil teljes tartománynevet és megbízható HTTPS-t igényel; a puszta IP-cím és a sima HTTP nem támogatott
2. **Állítsa be:** `RP_ID`
3. **Állítsa be:** `RP_ORIGINS` · HTTPS
4. **Állítsa be:** `http://127.0.0.1:6080`
5. **Tesztelje:** `GET /api/next/health`
6. **Jegyezze fel:** a passkey regisztrálása előtt válassza ki az állomásnevet; későbbi módosítása új regisztrációt igényel, a localhost pedig csak helyi beállítási kivétel

### Parancsok és beállítások

#### Mindkét csatorna

```text
RP_ID=discvault.example.com
RP_ORIGINS=https://discvault.example.com
curl --fail https://discvault.example.com/api/next/health
```
