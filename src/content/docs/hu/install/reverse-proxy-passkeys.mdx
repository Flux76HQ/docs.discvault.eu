## Eljárás

1. **Állítsa be:** `RP_ID`
2. **Állítsa be:** `RP_ORIGINS` · HTTPS
3. **Állítsa be:** `http://127.0.0.1:6080`
4. **Tesztelje:** `GET /api/next/health`

### Parancsok és beállítások

#### Mindkét csatorna

