## Eljárás

1. **Válassza ki:** FQDN · a localhost kivételén kívül a passkey stabil teljes tartománynevet és megbízható HTTPS-t igényel; a puszta IP-cím és a sima HTTP nem támogatott
2. **Válassza ki:** [Telepítés: Legacy Authentication](/hu/install/legacy-authentication/) · FQDN nélküli közvetlen privát vagy loopback IP esetén hagyja üresen az `RP_ID` és `RP_ORIGINS` értékét
3. **Állítsa be:** `RP_ID`
4. **Állítsa be:** `RP_ORIGINS` · HTTPS
5. **Állítsa be:** `6080` `6180`
6. **Tesztelje:** `GET /api/next/health`
7. **Jegyezze fel:** a passkey regisztrálása előtt válassza ki az állomásnevet; későbbi módosítása új regisztrációt igényel, a localhost pedig csak helyi beállítási kivétel

### Parancsok és beállítások

#### DiscVault v26 – stabil

```text
RP_ID=discvault.example.com
RP_NAME=DiscVault
RP_ORIGINS=https://discvault.example.com
curl --fail https://discvault.example.com/api/next/health
```

#### DiscVault v26 – béta

```text
RP_ID=discvault.example.com
RP_NAME=DiscVault
RP_ORIGINS=https://discvault.example.com
LEGACY_AUTH_ENABLED=true
curl --fail http://localhost:6180/api/next/health
```
