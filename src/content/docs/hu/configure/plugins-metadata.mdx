## Eljárás

1. **Ellenőrizze:** `GET /api/next/plugins/registry`
2. **Állítsa be:** Adminisztráció → Bővítmények → Szolgáltató → Beállítás · titokként tárolt szolgáltatói hitelesítő adat
3. **Tesztelje:** Adminisztráció → Bővítmények → Állapotellenőrzés · Adminisztráció → Bővítmények → Felderítés mentés nélkül · `dryRun`
4. **Jegyezze fel:** `secret app_settings`

### Parancsok és beállítások

#### Mindkét csatorna

```bash
curl --fail http://localhost:6080/api/next/plugins/registry
```
