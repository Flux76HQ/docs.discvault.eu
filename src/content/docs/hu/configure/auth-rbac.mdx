## Eljárás

1. **Nyissa meg:** Adminisztráció → Biztonság → Hitelesítés engedélyezése
2. **Hozza létre:** Adminisztráció → Felhasználók → 48 órás meghívó létrehozása
3. **Állítsa be:** Adminisztráció → Szerepkörök → Alap / Speciális
4. **Tesztelje:** `48 h`
