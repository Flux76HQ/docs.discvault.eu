## Eljárás

1. **Nyissa meg:** Könyvtár → Keresés
2. **Állítsa be:** Könyvtár → Szűrő → Formátum / Műfaj / Csoport
3. **Hasonlítsa össze:** Könyvtár → Nézet → Poszterek / Lista
4. **Tesztelje:** Cím → Részletek → Megnézendő / Megnézett
5. **Nyissa meg:** Beállítások → Preferenciák → Collectors → Collectors mode / Merge multiple editions as one title
6. **Állítsa be:** a Collectors Mode alapértelmezésben ki van kapcsolva, és fejlett konténerfunkciókat tesz elérhetővé; a `merge_editions_as_title` külön, szintén alapértelmezésben kikapcsolt beállítás
7. **Tesztelje:** a `merge_editions_as_title` bekapcsolásakor több fizikai változat egy könyvtárcsempén jelenik meg; a csempét megnyitva minden kiadás látható
8. **Hasonlítsa össze:** a box-set saját vonalkóddal rendelkező fizikai kereskedelmi termék; filmjei külön címek maradnak közvetlen tagként összekapcsolva
9. **Hasonlítsa össze:** a vault felhasználó által létrehozott polc-, tárolódoboz- vagy helycsoport közvetlen filmtagokkal, kereskedelmi termék jelentése nélkül
10. **Hasonlítsa össze:** a collection tematikus csoport, amelynek rendezett elemei filmek, box-setek, vaultok vagy más collection csoportok lehetnek, így beágyazható
11. **Tesztelje:** a Collectors Mode kikapcsolása elrejti a konténermegjelenítést, de nem törli a konténereket vagy tagjaikat
