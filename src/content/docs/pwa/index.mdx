## Expected result

The PWA opens from the configured server and routes to focused usage guidance.

- Address bar → Install DiscVault · Share → Add to Home Screen · Browser menu → Install app
- cached screens may open, but fresh data and writes require a network connection

## Safety and rollback

:::caution
Browser installation does not alter server data; remove only the installed shell if needed.

- remove only the installed PWA shell; the server library and account data remain untouched
:::
