## Goal and result

Browse, search, filter, and update titles, containers, box sets, collections, watchlist, and history.

## Applies to

In both DiscVault v26 channels, available actions are permission-scoped; viewers cannot perform editor or administrator changes.

### DiscVault v26 stable

- DiscVault v26 stable · `ghcr.io/helmerznl/discvault:latest`
- PWA · RBAC · account permissions

### DiscVault v26 beta

- DiscVault v26 beta · `ghcr.io/helmerznl/discvault:beta`
- PWA · RBAC · account permissions

## Prerequisites

- signed-in account with the required permission
- collection permission

## Procedure

1. **Open:** Library → Search · known title
2. **Configure:** Library → Filter → Format / Genre / Group
3. **Compare:** Library → View → Posters / List · same result count
4. **Test:** Title → Details → Watchlist / Watched · reload persistence



## Expected result

Search and filters return expected records and an allowed watchlist change persists after reload.

- Library → Search
- Library → Filter → Format / Genre / Group
- Title → Details → Watchlist / Watched

## Safety and rollback

:::caution
Cancel edits before saving. For an incorrect saved change, restore the field in the UI; use a server backup only for bulk corruption.

- cancel an incorrect edit before saving, restore a saved field in the UI, and use a matching server backup for bulk corruption
:::
