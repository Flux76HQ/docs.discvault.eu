## Procedura

1. **Scegli:** [Installazione: Docker Compose](/it/install/docker-compose/) · DiscVault 26
2. **Scegli:** [Installazione: Docker run](/it/install/docker-run/)
3. **Scegli:** [Installazione: Unraid](/it/install/unraid/) · Unraid
4. **Scegli:** le nuove installazioni usano DiscVault v26 `:latest` o `:beta`; l’immagine congelata `:legacy` è solo per un’installazione esistente della generazione precedente non ancora pronta alla migrazione
5. **Conserva:** mantieni invariati volumi, variabili d’ambiente, porte e percorso dati della generazione precedente; non sostituire mai un’immagine nella topologia Compose v26 PostgreSQL con `:legacy`
6. **Apri:** [Aggiornamento e migrazione: Migrazione a DiscVault 26](/it/update/migration/) · → DiscVault 26
7. **Registra:** `DISCVAULT_IMAGE` · solo per un deployment esistente o legacy: set di backup corrispondente
