## Prerequisiti

- credenziale del provider salvata come segreto

- sessione proprietario o amministratore

## Procedura

1. **Configura:** Amministrazione → Plugin → Provider → Configura · Plex Jellyfin
2. **Prova:** Amministrazione → Plugin → Controllo stato
3. **Esegui:** Amministrazione → Plugin → Scopri senza salvare · `dryRun=true`
4. **Confronta:** `/api/next/digital-items?limit=200`

### Comandi e impostazioni

curl --fail http://localhost:6080/api/next/digital-sources
