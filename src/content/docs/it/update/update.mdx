## Procedura

1. **Controlla:** le nuove installazioni usano DiscVault v26 `:latest` o `:beta`; l’immagine congelata `:legacy` è solo per un’installazione esistente della generazione precedente non ancora pronta alla migrazione
2. **Esegui il backup:** set di backup corrispondente
3. **Scegli:** Solo ramo Legacy: dopo il backup cambia soltanto il tag immagine del deployment esistente della vecchia generazione in `:legacy`, scaricalo e ricrealo senza modificarne la topologia, verifica la collezione e fermati; non passare questo deployment esistente alla topologia v26 in questa pagina
4. **Apri:** [Aggiornamento e migrazione: Migrazione a DiscVault 26](/it/update/migration/) · DiscVault 26
5. **Conserva:** mantieni invariati volumi, variabili d’ambiente, porte e percorso dati della generazione precedente; non sostituire mai un’immagine nella topologia Compose v26 PostgreSQL con `:legacy`
6. **Verifica:** `DiscVault v26` · `PREVIOUS_IMAGE=@sha256:…`
7. **Scegli:** `DiscVault v26` · `DISCVAULT_IMAGE` · `:latest` `:beta`
8. **Esegui:** `DiscVault v26` · `--force-recreate`
9. **Confronta:** `DiscVault v26` · è in esecuzione il digest previsto, lo stato è integro e accesso proprietario, conteggio e campione multimediale riescono

### Comandi e impostazioni

#### DiscVault v26 stabile

```bash
export DISCVAULT_IMAGE=ghcr.io/helmerznl/discvault:latest
```

#### DiscVault v26 beta

```bash
export DISCVAULT_IMAGE=ghcr.io/helmerznl/discvault:beta
```

#### Entrambi i canali

```bash
set -euo pipefail
test -n "$DISCVAULT_IMAGE"
case "$DISCVAULT_IMAGE" in
  ghcr.io/helmerznl/discvault:latest|ghcr.io/helmerznl/discvault:beta) ;;
  *) printf '%s\n' "$DISCVAULT_IMAGE" >&2; exit 64 ;;
esac
cd /opt/discvault
ENV_FILE=/opt/discvault/.env
ENV_TMP="$(mktemp "$ENV_FILE.update.XXXXXX")"
trap 'rm -f "$ENV_TMP"' EXIT
awk -v image="$DISCVAULT_IMAGE" '
  /^DISCVAULT_IMAGE=/ { print "DISCVAULT_IMAGE=" image; found=1; next }
  { print }
  END { if (!found) exit 42 }
' "$ENV_FILE" > "$ENV_TMP"
chmod 0600 "$ENV_TMP"
docker compose --env-file "$ENV_TMP" -p discvault -f compose.yaml config --quiet
docker compose --env-file "$ENV_TMP" -p discvault -f compose.yaml pull next-api next-worker next-mcp
docker compose --env-file "$ENV_TMP" -p discvault -f compose.yaml up -d --force-recreate
docker compose --env-file "$ENV_TMP" -p discvault -f compose.yaml ps
curl --fail http://localhost:6080/api/next/health
mv "$ENV_TMP" "$ENV_FILE"
trap - EXIT
```
