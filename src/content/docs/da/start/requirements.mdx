## Fremgangsmåde

1. **Kontrollér:** Docker Engine `≥24` · Docker Compose `v2`
2. **Kontrollér:** `≥2 GB` · skrivbart vedvarende lager og ledig backupplads
3. **Konfigurér:** DNS · TLS · HTTPS · `RP_ID` · `RP_ORIGINS`
4. **Registrér:** valgt udgivelseskanal
