## Forudsætninger

- ejer- eller administratorsession
- udbyderlegitimationsoplysning gemt som hemmelighed

## Fremgangsmåde

1. **Kontrollér:** `GET /api/next/plugins/registry`
2. **Konfigurér:** Administration → Plugins → Udbyder → Konfigurér · udbyderlegitimationsoplysning gemt som hemmelighed
3. **Test:** Administration → Plugins → Tilstandskontrol · Administration → Plugins → Find uden at gemme · `dryRun`
4. **Registrér:** `secret app_settings`

### Kommandoer og indstillinger

#### Begge kanaler

```bash
curl --fail http://localhost:6080/api/next/plugins/registry
curl --fail http://localhost:6080/api/next/metadata/plugins
```
