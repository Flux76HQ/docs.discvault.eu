## Fremgangsmåde

1. **Åbn:** Administration → Sikkerhed → Aktivér godkendelse
2. **Opret:** Administration → Brugere → Opret 48-timers invitation
3. **Konfigurér:** Administration → Roller → Grundlæggende / Avanceret
4. **Test:** `48 h`

### Kommandoer og indstillinger

#### Begge kanaler

```bash
curl --fail http://localhost:6080/api/next/auth/status
curl --fail http://localhost:6080/api/next/auth/rbac
```
