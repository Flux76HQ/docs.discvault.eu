## Fremgangsmåde

1. **Konfigurér:** `RP_ID`
2. **Konfigurér:** `RP_ORIGINS` · HTTPS
3. **Konfigurér:** `http://127.0.0.1:6080`
4. **Test:** `GET /api/next/health`

### Kommandoer og indstillinger

#### Begge kanaler

```text

RP_ORIGINS=https://discvault.example.com
curl --fail https://discvault.example.com/api/next/health
```
