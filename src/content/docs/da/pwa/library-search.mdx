## Fremgangsmåde

1. **Åbn:** Bibliotek → Søg
2. **Konfigurér:** Bibliotek → Filter → Format / Genre / Gruppe
3. **Sammenlign:** Bibliotek → Visning → Plakater / Liste
4. **Test:** Titel → Detaljer → Se senere / Set
5. **Åbn:** Indstillinger → Præferencer → Collectors → Collectors mode / Merge multiple editions as one title
6. **Konfigurér:** Collectors Mode er slået fra som standard og viser avancerede containerfunktioner; `merge_editions_as_title` er en separat indstilling, som også er slået fra som standard
7. **Test:** når `merge_editions_as_title` er aktiveret, vises flere fysiske versioner som én biblioteksflise; åbn flisen for at se alle udgaver
8. **Sammenlign:** et box-set er et fysisk detailprodukt med egen stregkode; filmene forbliver individuelle titler knyttet som direkte filmmedlemmer
9. **Sammenlign:** en vault er en brugeroprettet hylde-, opbevaringskasse- eller placeringsgruppe med direkte filmmedlemmer og uden betydning som detailprodukt
10. **Sammenlign:** en collection er en tematisk gruppe, hvis ordnede elementer kan være film, box-sets, vaults eller andre collections, så indlejring understøttes
11. **Test:** deaktivering af Collectors Mode skjuler containerpræsentationen, men sletter ikke containere eller deres medlemmer

## Sikkerhed og tilbagerulning

:::caution
Den sikre gendannelsesgrænse for **PWA: Bibliotek og søgning** kræver disse handlinger, før noget startes eller fortsættes: deaktivering af Collectors Mode skjuler containerpræsentationen, men sletter ikke containere eller deres medlemmer; annuller en forkert redigering før lagring, gendan et gemt felt i UI, og brug en passende serverbackup ved omfattende skade. Sidespecifikke kontrolpunkter: PWA · `watchlist` · `history` · RBAC · `merge_editions_as_title` · `box_set` · `vault` · `collection`.
