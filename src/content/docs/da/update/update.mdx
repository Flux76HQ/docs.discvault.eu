## Fremgangsmåde

1. **Kontrollér:** nye installationer bruger DiscVault v26 `:latest` eller `:beta`; det frosne `:legacy`-image er kun til en eksisterende installation af den forrige generation, som endnu ikke er klar til migrering
2. **Sikkerhedskopiér:** tilsvarende backupsæt
3. **Vælg:** Kun Legacy-grenen: efter backup ændres kun image-tagget for den eksisterende gamle installation til `:legacy`; hent og genskab den med uændret topologi, kontrollér samlingen, og stop; kør ikke v26-kommandoerne nedenfor
4. **Åbn:** [Opdatering og migrering: Migrering til DiscVault 26](/da/update/migration/) · DiscVault 26
5. **Bevar:** behold den forrige generations diskenheder, miljøvariabler, porte og datasti uændret; erstat aldrig et v26 PostgreSQL Compose-image med `:legacy`
6. **Bekræft:** `DiscVault v26` · `PREVIOUS_IMAGE=@sha256:…`
7. **Vælg:** `DiscVault v26` · `DISCVAULT_IMAGE` · `:latest` `:beta`
8. **Kør:** `DiscVault v26` · `--force-recreate`
9. **Sammenlign:** `DiscVault v26` · det tilsigtede image-digest kører, sundhed lykkes, og ejerlogin, bibliotekstal og én medieprøve består

### Kommandoer og indstillinger

#### DiscVault v26 – stabil

```bash
export DISCVAULT_IMAGE=ghcr.io/helmerznl/discvault:latest
```

#### DiscVault v26 – beta

```bash
export DISCVAULT_IMAGE=ghcr.io/helmerznl/discvault:beta
```

#### Begge kanaler

```bash
set -euo pipefail
test -n "$DISCVAULT_IMAGE"
case "$DISCVAULT_IMAGE" in
  ghcr.io/helmerznl/discvault:latest|ghcr.io/helmerznl/discvault:beta) ;;
  *) printf '%s\n' "$DISCVAULT_IMAGE" >&2; exit 64 ;;
esac
cd /opt/discvault
ENV_FILE=/opt/discvault/.env
ENV_TMP="$(mktemp "$ENV_FILE.update.XXXXXX")"
trap 'rm -f "$ENV_TMP"' EXIT
awk -v image="$DISCVAULT_IMAGE" '
  /^DISCVAULT_IMAGE=/ { print "DISCVAULT_IMAGE=" image; found=1; next }
  { print }
  END { if (!found) exit 42 }
' "$ENV_FILE" > "$ENV_TMP"
chmod 0600 "$ENV_TMP"
docker compose --env-file "$ENV_TMP" -p discvault -f compose.yaml config --quiet
docker compose --env-file "$ENV_TMP" -p discvault -f compose.yaml pull next-api next-worker next-mcp
docker compose --env-file "$ENV_TMP" -p discvault -f compose.yaml up -d --force-recreate
docker compose --env-file "$ENV_TMP" -p discvault -f compose.yaml ps
curl --fail http://localhost:6080/api/next/health
mv "$ENV_TMP" "$ENV_FILE"
trap - EXIT
```
