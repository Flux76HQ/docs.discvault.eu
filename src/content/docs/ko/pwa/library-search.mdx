## 절차

1. **열기:** 라이브러리 → 검색
2. **구성:** 라이브러리 → 필터 → 형식 / 장르 / 그룹
3. **비교:** 라이브러리 → 보기 → 포스터 / 목록
4. **테스트:** 제목 → 상세 → 관심 목록 / 시청함
5. **열기:** 설정 → 환경설정 → Collectors → Collectors mode / Merge multiple editions as one title
6. **구성:** Collectors Mode는 기본적으로 꺼져 있고 고급 컨테이너 기능을 표시하며 `merge_editions_as_title`은 역시 기본적으로 꺼진 별도 설정임
7. **테스트:** `merge_editions_as_title`을 켜면 여러 실제 버전이 하나의 라이브러리 타일로 표시되고 타일을 열어 모든 에디션을 볼 수 있음
8. **비교:** box-set은 자체 바코드가 있는 실제 소매 제품이며 영화는 직접 영화 구성원으로 연결된 개별 타이틀로 유지됨
9. **비교:** vault는 사용자가 만든 선반, 보관 상자 또는 위치 그룹으로 직접 영화 구성원을 가지며 소매 제품을 뜻하지 않음
10. **비교:** collection은 순서가 지정된 항목으로 영화, box-set, vault 또는 다른 collection을 포함할 수 있는 주제별 그룹이므로 중첩을 지원함
11. **테스트:** Collectors Mode를 끄면 컨테이너 표시가 숨겨지지만 컨테이너나 구성원은 삭제되지 않음
