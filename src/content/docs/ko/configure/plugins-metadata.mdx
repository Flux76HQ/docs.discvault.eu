## 절차

1. **검사:** `GET /api/next/plugins/registry`
2. **구성:** 관리 → 플러그인 → 공급자 → 구성 · 비밀로 저장되는 공급자 자격 증명
3. **테스트:** 관리 → 플러그인 → 상태 확인 · 관리 → 플러그인 → 저장 없이 검색 · `dryRun`
4. **기록:** `secret app_settings`

### 명령 및 설정
