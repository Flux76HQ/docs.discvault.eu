## 前提条件

- 所有者または管理者セッション
- シークレットとして保存するプロバイダー資格情報

## 手順

1. **確認:** `GET /api/next/plugins/registry`
2. **設定:** 管理 → プラグイン → プロバイダー → 設定 · シークレットとして保存するプロバイダー資格情報
3. **テスト:** 管理 → プラグイン → 状態確認 · 管理 → プラグイン → 保存せずに検出 · `dryRun`
4. **記録:** `secret app_settings`

### コマンドと設定

#### 両方のチャンネル

```bash
curl --fail http://localhost:6080/api/next/plugins/registry
curl --fail http://localhost:6080/api/next/metadata/plugins
```
