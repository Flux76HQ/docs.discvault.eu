## 前提条件

## 手順

1. **開く:** ライブラリ → 検索
2. **設定:** ライブラリ → フィルター → 形式 / ジャンル / グループ
3. **比較:** ライブラリ → 表示 → ポスター / リスト
4. **テスト:** タイトル → 詳細 → ウォッチリスト / 視聴済み
5. **設定:** Collectors Mode は既定でオフで、高度なコンテナー機能を表示します。`merge_editions_as_title` は別の設定で、これも既定ではオフです
6. **テスト:** `merge_editions_as_title` を有効にすると複数の物理版が 1 つのライブラリタイルに表示され、タイルを開くとすべての版を確認できます
7. **比較:** box-set は固有のバーコードを持つ物理的な販売商品です。収録映画は個別タイトルのまま、直接の映画メンバーとして関連付けられます
8. **比較:** vault はユーザーが作る棚、収納箱、場所のグループで、映画を直接メンバーに持ち、小売商品の意味はありません
9. **比較:** collection はテーマ別グループで、順序付き項目として映画、box-set、vault、別の collection を含められるため、入れ子に対応します
10. **テスト:** Collectors Mode をオフにするとコンテナー表示は隠れますが、コンテナーやメンバーは削除されません
