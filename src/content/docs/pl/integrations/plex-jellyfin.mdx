## Procedura

1. **Skonfiguruj:** Administracja → Wtyczki → Dostawca → Konfiguruj · Plex Jellyfin
2. **Przetestuj:** Administracja → Wtyczki → Kontrola stanu
3. **Uruchom:** Administracja → Wtyczki → Wykryj bez zapisywania · `dryRun=true`
4. **Porównaj:** `/api/next/digital-items?limit=200`
