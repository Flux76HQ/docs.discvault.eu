## Wymagania wstępne

- wybrany kanał wydań
- dzienniki ze znacznikiem czasu i ukrytymi danymi
- `postgres`

## Procedura

1. **Zapisz:** `docker compose`
2. **Odizoluj:** kontener → odwrotne proxy → przeglądarka/aplikacja → dostawca → dane
3. **Sprawdź:** dzienniki ze znacznikiem czasu i ukrytymi danymi
4. **Zachowaj:** PostgreSQL `/data`

### Polecenia i ustawienia

#### Oba kanały

```bash
docker compose --env-file /opt/discvault/.env -p discvault -f /opt/discvault/compose.yaml ps
curl --fail http://localhost:6080/api/next/health
docker compose --env-file /opt/discvault/.env -p discvault -f /opt/discvault/compose.yaml logs --tail=200 next-api next-worker next-mcp postgres
```
