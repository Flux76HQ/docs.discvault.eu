## Wymagania wstępne

- dostęp właściciela

- druga metoda odzyskiwania

## Procedura

1. **Otwórz:** Administracja → Bezpieczeństwo → Włącz uwierzytelnianie
2. **Utwórz:** Administracja → Użytkownicy → Utwórz zaproszenie na 48 godzin
3. **Skonfiguruj:** Administracja → Role → Podstawowe / Zaawansowane
4. **Przetestuj:** `48 h`

### Polecenia i ustawienia

#### Oba kanały

```bash
curl --fail http://localhost:6080/api/next/auth/status
curl --fail http://localhost:6080/api/next/auth/rbac
```
