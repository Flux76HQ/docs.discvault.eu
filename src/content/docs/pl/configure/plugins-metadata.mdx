## Wymagania wstępne

- sesja właściciela lub administratora
- poświadczenie dostawcy zapisane jako sekret

## Procedura

1. **Sprawdź:** `GET /api/next/plugins/registry`
2. **Skonfiguruj:** Administracja → Wtyczki → Dostawca → Konfiguruj · poświadczenie dostawcy zapisane jako sekret
3. **Przetestuj:** Administracja → Wtyczki → Kontrola stanu · Administracja → Wtyczki → Wykryj bez zapisywania · `dryRun`
4. **Zapisz:** `secret app_settings`

### Polecenia i ustawienia

#### Oba kanały

```bash
curl --fail http://localhost:6080/api/next/plugins/registry
curl --fail http://localhost:6080/api/next/metadata/plugins
```
