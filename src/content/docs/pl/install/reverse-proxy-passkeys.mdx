## Procedura

1. **Skonfiguruj:** `RP_ID`
2. **Skonfiguruj:** `RP_ORIGINS` · HTTPS
3. **Skonfiguruj:** `http://127.0.0.1:6080`
4. **Przetestuj:** `GET /api/next/health`

### Polecenia i ustawienia

#### Oba kanały

```text
RP_ID=discvault.example.com
RP_ORIGINS=https://discvault.example.com
curl --fail https://discvault.example.com/api/next/health
```
