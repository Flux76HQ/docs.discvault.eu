## Procedura

1. **Otwórz:** Biblioteka → Szukaj
2. **Skonfiguruj:** Biblioteka → Filtr → Format / Gatunek / Grupa
3. **Porównaj:** Biblioteka → Widok → Plakaty / Lista
4. **Przetestuj:** Tytuł → Szczegóły → Do obejrzenia / Obejrzane
