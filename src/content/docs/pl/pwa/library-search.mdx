## Procedura

1. **Otwórz:** Biblioteka → Szukaj
2. **Skonfiguruj:** Biblioteka → Filtr → Format / Gatunek / Grupa
3. **Porównaj:** Biblioteka → Widok → Plakaty / Lista
4. **Przetestuj:** Tytuł → Szczegóły → Do obejrzenia / Obejrzane
5. **Skonfiguruj:** Collectors Mode jest domyślnie wyłączony i udostępnia zaawansowane funkcje kontenerów; `merge_editions_as_title` to osobne ustawienie, również domyślnie wyłączone
6. **Przetestuj:** po włączeniu `merge_editions_as_title` wiele wersji fizycznych jest widocznych jako jeden kafelek biblioteki; otwórz go, aby zobaczyć wszystkie wydania
7. **Porównaj:** box-set to fizyczny produkt sklepowy z własnym kodem kreskowym; filmy pozostają osobnymi tytułami powiązanymi jako bezpośrednie elementy
8. **Porównaj:** vault to utworzona przez użytkownika grupa półki, pudełka lub lokalizacji z bezpośrednimi filmami, bez znaczenia produktu sklepowego
9. **Porównaj:** collection to grupa tematyczna, której uporządkowanymi elementami mogą być filmy, box-sety, vaulty lub inne collections, więc obsługuje zagnieżdżanie
10. **Przetestuj:** wyłączenie Collectors Mode ukrywa prezentację kontenerów, ale nie usuwa kontenerów ani ich elementów
