## Διαδικασία

1. **Ανοίξτε:** Βιβλιοθήκη → Αναζήτηση
2. **Ρυθμίστε:** Βιβλιοθήκη → Φίλτρο → Μορφή / Είδος / Ομάδα
3. **Συγκρίνετε:** Βιβλιοθήκη → Προβολή → Αφίσες / Λίστα
4. **Δοκιμάστε:** Τίτλος → Λεπτομέρειες → Λίστα παρακολούθησης / Προβλήθηκε
5. **Ρυθμίστε:** το Collectors Mode είναι απενεργοποιημένο από προεπιλογή και εμφανίζει προηγμένες λειτουργίες containers· το `merge_editions_as_title` είναι ξεχωριστή ρύθμιση, επίσης απενεργοποιημένη από προεπιλογή
6. **Δοκιμάστε:** όταν ενεργοποιείται το `merge_editions_as_title`, πολλές φυσικές εκδόσεις εμφανίζονται ως ένα πλακίδιο βιβλιοθήκης· ανοίξτε το για να δείτε κάθε έκδοση
7. **Συγκρίνετε:** ένα box-set είναι φυσικό εμπορικό προϊόν με δικό του barcode· οι ταινίες του παραμένουν ξεχωριστοί τίτλοι συνδεδεμένοι ως άμεσα μέλη
8. **Συγκρίνετε:** ένα vault είναι ομαδοποίηση ραφιού, κουτιού αποθήκευσης ή τοποθεσίας που δημιουργεί ο χρήστης, με άμεσα μέλη ταινιών και χωρίς έννοια εμπορικού προϊόντος
9. **Συγκρίνετε:** ένα collection είναι θεματική ομαδοποίηση της οποίας τα ταξινομημένα στοιχεία μπορεί να είναι ταινίες, box-sets, vaults ή άλλα collections, επομένως υποστηρίζεται εμφώλευση
10. **Δοκιμάστε:** η απενεργοποίηση του Collectors Mode κρύβει την παρουσίαση containers αλλά δεν διαγράφει containers ή τα μέλη τους
