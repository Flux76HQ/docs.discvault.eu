## Διαδικασία

1. **Ελέγξτε:** οι νέες εγκαταστάσεις χρησιμοποιούν DiscVault v26 `:latest` ή `:beta`· η παγωμένη εικόνα `:legacy` προορίζεται μόνο για υπάρχουσα εγκατάσταση προηγούμενης γενιάς που δεν είναι έτοιμη για μετεγκατάσταση
2. **Δημιουργήστε αντίγραφο:** αντίστοιχο σύνολο αντιγράφων
3. **Επιλέξτε:** Μόνο κλάδος Legacy: μετά το αντίγραφο αλλάξτε μόνο το tag εικόνας του υπάρχοντος παλιού deployment σε `:legacy`, κάντε pull και αναδημιουργία με αμετάβλητη τοπολογία, επαληθεύστε τη συλλογή και σταματήστε· μην εκτελέσετε τις παρακάτω εντολές v26
4. **Ανοίξτε:** [Ενημέρωση και μετεγκατάσταση: Μετεγκατάσταση στο DiscVault 26](/el/update/migration/) · DiscVault 26
5. **Διατηρήστε:** διατηρήστε αμετάβλητα τα volumes, τις μεταβλητές περιβάλλοντος, τις θύρες και τη διαδρομή δεδομένων της προηγούμενης γενιάς· μην αντικαθιστάτε ποτέ εικόνα στην τοπολογία v26 PostgreSQL Compose με `:legacy`
6. **Επαληθεύστε:** `DiscVault v26` · `PREVIOUS_IMAGE=@sha256:…`
7. **Επιλέξτε:** `DiscVault v26` · `DISCVAULT_IMAGE` · `:latest` `:beta`
8. **Εκτελέστε:** `DiscVault v26` · `--force-recreate`
9. **Συγκρίνετε:** `DiscVault v26` · εκτελείται το σωστό digest εικόνας, ο έλεγχος υγείας πετυχαίνει και περνούν σύνδεση κατόχου, πλήθος βιβλιοθήκης και δείγμα πολυμέσων

### Εντολές και ρυθμίσεις

#### DiscVault v26 – σταθερό

```bash
export DISCVAULT_IMAGE=ghcr.io/helmerznl/discvault:latest
```

#### DiscVault v26 – beta

```bash
export DISCVAULT_IMAGE=ghcr.io/helmerznl/discvault:beta
```

#### Και τα δύο κανάλια

```bash
set -euo pipefail
test -n "$DISCVAULT_IMAGE"
case "$DISCVAULT_IMAGE" in
  ghcr.io/helmerznl/discvault:latest|ghcr.io/helmerznl/discvault:beta) ;;
  *) printf '%s\n' "$DISCVAULT_IMAGE" >&2; exit 64 ;;
esac
cd /opt/discvault
ENV_FILE=/opt/discvault/.env
ENV_TMP="$(mktemp "$ENV_FILE.update.XXXXXX")"
trap 'rm -f "$ENV_TMP"' EXIT
awk -v image="$DISCVAULT_IMAGE" '
  /^DISCVAULT_IMAGE=/ { print "DISCVAULT_IMAGE=" image; found=1; next }
  { print }
  END { if (!found) exit 42 }
' "$ENV_FILE" > "$ENV_TMP"
chmod 0600 "$ENV_TMP"
docker compose --env-file "$ENV_TMP" -p discvault -f compose.yaml config --quiet
docker compose --env-file "$ENV_TMP" -p discvault -f compose.yaml pull next-api next-worker next-mcp
docker compose --env-file "$ENV_TMP" -p discvault -f compose.yaml up -d --force-recreate
docker compose --env-file "$ENV_TMP" -p discvault -f compose.yaml ps
curl --fail http://localhost:6080/api/next/health
mv "$ENV_TMP" "$ENV_FILE"
trap - EXIT
```
