## Διαδικασία

1. **Ρυθμίστε:** `RP_ID`
2. **Ρυθμίστε:** `RP_ORIGINS` · HTTPS
3. **Ρυθμίστε:** `http://127.0.0.1:6080`
4. **Δοκιμάστε:** `GET /api/next/health`

### Εντολές και ρυθμίσεις

#### Και τα δύο κανάλια

```text

RP_ORIGINS=https://discvault.example.com

```
