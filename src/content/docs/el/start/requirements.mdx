## Στόχος και αποτέλεσμα

Ολοκληρώστε τη ροή εργασίας της σελίδας για **Έναρξη: Επισκόπηση**. Η διαδικασία χρησιμοποιεί Docker Engine `24+` · Docker Compose `v2` · PostgreSQL `17` · HTTPS.

## Ισχύει για

Αυτή η σελίδα ισχύει μόνο για **Έναρξη: Επισκόπηση** (όλες οι πλατφόρμες). Τα όρια καναλιού και δεδομένων είναι: DiscVault · `ghcr.io/helmerznl/discvault:latest`; Docker Compose `v2` · PostgreSQL `17` · `/data`; DiscVault · `ghcr.io/helmerznl/discvault:beta`; Docker Compose `v2` · PostgreSQL `17` · `/data`. Σημεία ελέγχου της σελίδας: Docker Engine `24+` · Docker Compose `v2` · PostgreSQL `17` · HTTPS.

## Προϋποθέσεις

- Docker Engine `24+`
- εγγράψιμη μόνιμη αποθήκευση και ελεύθερος χώρος αντιγράφων
- DNS TLS

## Διαδικασία

1. **Ελέγξτε:** Docker Engine `≥24` · Docker Compose `v2`
2. **Ελέγξτε:** `≥2 GB` · εγγράψιμη μόνιμη αποθήκευση και ελεύθερος χώρος αντιγράφων
3. **Ρυθμίστε:** DNS · TLS · HTTPS · `RP_ID` · `RP_ORIGINS`
4. **Καταγράψτε:** επιλεγμένο κανάλι έκδοσης
