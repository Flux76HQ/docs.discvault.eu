## Προϋποθέσεις

- επιλεγμένο κανάλι έκδοσης
- αρχεία καταγραφής με χρόνο και απόκρυψη ιδιωτικών δεδομένων
- `postgres`

## Διαδικασία

1. **Καταγράψτε:** `docker compose`
2. **Απομονώστε:** περιέκτης → αντίστροφος διακομιστής μεσολάβησης → πρόγραμμα περιήγησης/εφαρμογή → πάροχος → δεδομένα
3. **Ελέγξτε:** αρχεία καταγραφής με χρόνο και απόκρυψη ιδιωτικών δεδομένων
4. **Διατηρήστε:** PostgreSQL `/data`

### Εντολές και ρυθμίσεις

#### Και τα δύο κανάλια

```bash
docker compose --env-file /opt/discvault/.env -p discvault -f /opt/discvault/compose.yaml ps
curl --fail http://localhost:6080/api/next/health
docker compose --env-file /opt/discvault/.env -p discvault -f /opt/discvault/compose.yaml logs --tail=200 next-api next-worker next-mcp postgres
```
