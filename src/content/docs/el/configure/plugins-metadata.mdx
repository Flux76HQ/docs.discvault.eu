## Διαδικασία

1. **Ελέγξτε:** `GET /api/next/plugins/registry`
2. **Ρυθμίστε:** Διαχείριση → Πρόσθετα → Πάροχος → Ρύθμιση · διαπιστευτήριο παρόχου αποθηκευμένο ως μυστικό
3. **Δοκιμάστε:** Διαχείριση → Πρόσθετα → Έλεγχος λειτουργίας · Διαχείριση → Πρόσθετα → Εντοπισμός χωρίς αποθήκευση · `dryRun`
4. **Καταγράψτε:** `secret app_settings`
