## Διαδικασία

1. **Ανοίξτε:** Διαχείριση → Ασφάλεια → Ενεργοποίηση ταυτοποίησης
2. **Δημιουργήστε:** Διαχείριση → Χρήστες → Δημιουργία πρόσκλησης 48 ωρών
3. **Ρυθμίστε:** Διαχείριση → Ρόλοι → Βασικό / Προχωρημένο
4. **Δοκιμάστε:** `48 h`
