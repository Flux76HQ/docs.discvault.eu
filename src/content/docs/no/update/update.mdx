## Fremgangsmåte

1. **Kontroller:** nye installasjoner bruker DiscVault v26 `:latest` eller `:beta`; det fryste `:legacy`-imaget er bare for en eksisterende installasjon av forrige generasjon som ennå ikke er klar for migrering
2. **Sikkerhetskopier:** tilhørende sikkerhetskopisett
3. **Velg:** Bare Legacy-grenen: etter sikkerhetskopien endrer du kun image-taggen for den eksisterende gamle installasjonen til `:legacy`, henter og oppretter den på nytt med uendret topologi, kontrollerer samlingen og stopper; ikke kjør v26-kommandoene nedenfor
4. **Åpne:** [Oppdatering og migrering: Migrering til DiscVault 26](/no/update/migration/) · DiscVault 26
5. **Bevar:** behold forrige generasjons volumer, miljøvariabler, porter og databane uendret; bytt aldri et image i v26 PostgreSQL Compose-topologien til `:legacy`
6. **Bekreft:** `DiscVault v26` · `PREVIOUS_IMAGE=@sha256:…`
7. **Velg:** `DiscVault v26` · `DISCVAULT_IMAGE` · `:latest` `:beta`
8. **Kjør:** `DiscVault v26` · `--force-recreate`
9. **Sammenlign:** `DiscVault v26` · tiltenkt image-digest kjører, helsekontrollen lykkes, og eierinnlogging, bibliotektall og ett medieutvalg består

### Kommandoer og innstillinger

#### DiscVault v26 – stabil

```bash
export DISCVAULT_IMAGE=ghcr.io/helmerznl/discvault:latest
```

#### DiscVault v26 – beta

```bash
export DISCVAULT_IMAGE=ghcr.io/helmerznl/discvault:beta
```

#### Begge kanaler

```bash
set -euo pipefail
test -n "$DISCVAULT_IMAGE"
case "$DISCVAULT_IMAGE" in
  ghcr.io/helmerznl/discvault:latest|ghcr.io/helmerznl/discvault:beta) ;;
  *) printf '%s\n' "$DISCVAULT_IMAGE" >&2; exit 64 ;;
esac
cd /opt/discvault
ENV_FILE=/opt/discvault/.env
ENV_TMP="$(mktemp "$ENV_FILE.update.XXXXXX")"
trap 'rm -f "$ENV_TMP"' EXIT
awk -v image="$DISCVAULT_IMAGE" '
  /^DISCVAULT_IMAGE=/ { print "DISCVAULT_IMAGE=" image; found=1; next }
  { print }
  END { if (!found) exit 42 }
' "$ENV_FILE" > "$ENV_TMP"
chmod 0600 "$ENV_TMP"
docker compose --env-file "$ENV_TMP" -p discvault -f compose.yaml config --quiet
docker compose --env-file "$ENV_TMP" -p discvault -f compose.yaml pull next-api next-worker next-mcp
docker compose --env-file "$ENV_TMP" -p discvault -f compose.yaml up -d --force-recreate
docker compose --env-file "$ENV_TMP" -p discvault -f compose.yaml ps
curl --fail http://localhost:6080/api/next/health
mv "$ENV_TMP" "$ENV_FILE"
trap - EXIT
```
