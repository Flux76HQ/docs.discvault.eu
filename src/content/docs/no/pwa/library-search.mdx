## Fremgangsmåte

1. **Åpne:** Bibliotek → Søk
2. **Konfigurer:** Bibliotek → Filter → Format / Sjanger / Gruppe
3. **Sammenlign:** Bibliotek → Visning → Plakater / Liste
4. **Test:** Tittel → Detaljer → Se senere / Sett
5. **Konfigurer:** Collectors Mode er av som standard og viser avanserte containerfunksjoner; `merge_editions_as_title` er en separat innstilling som også er av som standard
6. **Test:** når `merge_editions_as_title` er aktivert, vises flere fysiske versjoner som én bibliotekflis; åpne flisen for å se alle utgaver
7. **Sammenlign:** et box-set er et fysisk butikkprodukt med egen strekkode; filmene forblir individuelle titler koblet som direkte filmmedlemmer
8. **Sammenlign:** et vault er en brukeropprettet hylle-, oppbevaringsboks- eller plasseringsgruppe med direkte filmmedlemmer og uten betydning som butikkprodukt
9. **Sammenlign:** en collection er en tematisk gruppe der ordnede elementer kan være filmer, box-set, vaults eller andre collections, slik at nesting støttes
10. **Test:** å slå av Collectors Mode skjuler containervisningen, men sletter ikke containere eller medlemmer

## Sikkerhet og tilbakerulling

:::caution
Den sikre gjenopprettingsgrensen for **PWA: Bibliotek og søk** krever disse handlingene før noe startes eller fortsetter: å slå av Collectors Mode skjuler containervisningen, men sletter ikke containere eller medlemmer; avbryt feil redigering før lagring, gjenopprett et lagret felt i brukergrensesnittet, og bruk tilhørende serverkopi ved omfattende skade.
