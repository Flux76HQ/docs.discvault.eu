## Fremgangsmåte

1. **Kontroller:** `GET /api/next/plugins/registry`
2. **Konfigurer:** Administrasjon → Programtillegg → Leverandør → Konfigurer · leverandørlegitimasjon lagret som hemmelighet
3. **Test:** Administrasjon → Programtillegg → Tilstandskontroll · Administrasjon → Programtillegg → Oppdag uten å lagre · `dryRun`
4. **Registrer:** `secret app_settings`

### Kommandoer og innstillinger

#### Begge kanaler

```bash
curl --fail http://localhost:6080/api/next/plugins/registry
curl --fail http://localhost:6080/api/next/metadata/plugins
```
