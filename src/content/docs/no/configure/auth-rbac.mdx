## Fremgangsmåte

1. **Åpne:** Administrasjon → Sikkerhet → Aktiver autentisering
2. **Opprett:** Administrasjon → Brukere → Opprett 48-timers invitasjon
3. **Konfigurer:** Administrasjon → Roller → Grunnleggende / Avansert
4. **Test:** `48 h`
