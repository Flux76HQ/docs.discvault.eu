## Goal and result

Set up the native offline-first iOS/iPadOS app from a verified distribution source.

## Applies to

Beta source implementation. Local SwiftData library, scanning, lists, details, notifications, and app lock are verified in source; public App Store availability is not assumed.

### Beta channel

- source-verified beta build; no public store availability claim
- SwiftData · offline-first

## Prerequisites

- iOS/iPadOS `17+`
- build from a verified distribution source
- camera permission

## Procedure

1. **Open:** Launch → Onboarding → Create local library
2. **Create:** Library → Add → Scan barcode / Enter manually · one test title
3. **Configure:** Settings → App Lock
4. **Test:** airplane mode · the test title remains after app restart and an offline launch



## Expected result

Onboarding completes and a manually added or scanned title remains after restarting the app.

- Launch → Onboarding → Create local library
- the test title remains after app restart and an offline launch
