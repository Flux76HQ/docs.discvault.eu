## Toimenpiteet

1. **Avaa:** Kirjasto → Haku
2. **Määritä:** Kirjasto → Suodatin → Formaatti / Laji / Ryhmä
3. **Vertaa:** Kirjasto → Näkymä → Julisteet / Luettelo
4. **Testaa:** Nimike → Tiedot → Katselulista / Katsottu
5. **Määritä:** Collectors Mode on oletuksena pois käytöstä ja tuo esiin edistyneet container-ominaisuudet; `merge_editions_as_title` on erillinen asetus, joka on myös oletuksena pois
6. **Testaa:** kun `merge_editions_as_title` on käytössä, useat fyysiset versiot näkyvät yhtenä kirjastoruutuna; avaa ruutu nähdäksesi kaikki versiot
7. **Vertaa:** box-set on fyysinen vähittäistuote omalla viivakoodilla; sen elokuvat säilyvät erillisinä nimikkeinä ja suorina jäseninä
8. **Vertaa:** vault on käyttäjän luoma hylly-, säilytyslaatikko- tai sijaintiryhmä, jossa elokuvat ovat suoria jäseniä eikä sillä ole vähittäistuotteen merkitystä
9. **Vertaa:** collection on temaattinen ryhmä, jonka järjestetyt kohteet voivat olla elokuvia, box-settejä, vaulteja tai muita collection-ryhmiä, joten sisäkkäisyys on tuettu
10. **Testaa:** Collectors Moden poistaminen käytöstä piilottaa container-näkymän mutta ei poista containereita tai niiden jäseniä



## Odotettu tulos

Hyväksy tulos kohteelle **PWA: Kirjasto ja haku** vain kun kaikki seuraavat tulokset toteutuvat: Kirjasto → Haku; Kirjasto → Suodatin → Formaatti / Laji / Ryhmä; Nimike → Tiedot → Katselulista / Katsottu; kun `merge_editions_as_title` on käytössä, useat fyysiset versiot näkyvät yhtenä kirjastoruutuna; avaa ruutu nähdäksesi kaikki versiot; box-set on fyysinen vähittäistuote omalla viivakoodilla; sen elokuvat säilyvät erillisinä nimikkeinä ja suorina jäseninä; vault on käyttäjän luoma hylly-, säilytyslaatikko- tai sijaintiryhmä, jossa elokuvat ovat suoria jäseniä eikä sillä ole vähittäistuotteen merkitystä; collection on temaattinen ryhmä, jonka järjestetyt kohteet voivat olla elokuvia, box-settejä, vaulteja tai muita collection-ryhmiä, joten sisäkkäisyys on tuettu. Sivukohtaiset tarkistuspisteet: PWA · `watchlist` · `history` · RBAC · `merge_editions_as_title` · `box_set` · `vault` · `collection`.

- Kirjasto → Haku
- Kirjasto → Suodatin → Formaatti / Laji / Ryhmä
- Nimike → Tiedot → Katselulista / Katsottu
- kun `merge_editions_as_title` on käytössä, useat fyysiset versiot näkyvät yhtenä kirjastoruutuna; avaa ruutu nähdäksesi kaikki versiot
- box-set on fyysinen vähittäistuote omalla viivakoodilla; sen elokuvat säilyvät erillisinä nimikkeinä ja suorina jäseninä
