## Toimenpiteet

1. **Valitse:** [Asennus: Docker Compose](/fi/install/docker-compose/) · DiscVault 26
2. **Valitse:** [Asennus: Docker run](/fi/install/docker-run/)
3. **Valitse:** [Asennus: Unraid](/fi/install/unraid/) · Unraid
4. **Valitse:** uudet asennukset käyttävät DiscVault v26 -tunnistetta `:latest` tai `:beta`; jäädytetty `:legacy`-image on vain olemassa olevalle edellisen sukupolven asennukselle, jota ei vielä siirretä
5. **Säilytä:** pidä edellisen sukupolven taltiot, ympäristömuuttujat, portit ja datapolku ennallaan; älä koskaan vaihda v26 PostgreSQL Compose -topologian imagea `:legacy`-imageksi
6. **Avaa:** [Päivitys ja siirto: Siirtyminen DiscVault 26:een](/fi/update/migration/) · → DiscVault 26
7. **Kirjaa:** `DISCVAULT_IMAGE` · vain olemassa olevalle tai legacy-käyttöönotolle: vastaava varmuuskopiosarja

## Turvallisuus ja palautus

:::caution
Turvallinen palautusraja kohteelle **Asennus: Asennusvaihtoehdot** edellyttää näitä toimia ennen käynnistystä tai jatkamista: Vain Legacy-haara: vaihda varmuuskopioinnin jälkeen vain nykyisen vanhan käyttöönoton image-tunnisteeksi `:legacy`, vedä ja luo se uudelleen topologiaa muuttamatta, tarkista kokoelma ja lopeta; älä vaihda tätä olemassa olevaa käyttöönottoa v26-topologiaan tällä sivulla; jäädytetty `:legacy`-image ei saa DiscVault v26 -ominaisuuksia, siirron parannuksia eikä tulevia käyttöliittymäpäivityksiä; vain olemassa olevalle tai legacy-käyttöönotolle: vastaava varmuuskopiosarja; julkaisutiedot ja kirjattu muuttumaton image-tiiviste. Sivukohtaiset tarkistuspisteet: `:latest` · `:beta` · `:legacy` · `DISCVAULT_IMAGE` · PostgreSQL `/data` · `/update/migration/`.

- Vain Legacy-haara: vaihda varmuuskopioinnin jälkeen vain nykyisen vanhan käyttöönoton image-tunnisteeksi `:legacy`, vedä ja luo se uudelleen topologiaa muuttamatta, tarkista kokoelma ja lopeta; älä vaihda tätä olemassa olevaa käyttöönottoa v26-topologiaan tällä sivulla
- jäädytetty `:legacy`-image ei saa DiscVault v26 -ominaisuuksia, siirron parannuksia eikä tulevia käyttöliittymäpäivityksiä
- vain olemassa olevalle tai legacy-käyttöönotolle: vastaava varmuuskopiosarja

:::
