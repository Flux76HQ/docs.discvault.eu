## Toimenpiteet

1. **Määritä:** Ylläpito → Liitännäiset → Tarjoaja → Määritä · Plex Jellyfin
2. **Testaa:** Ylläpito → Liitännäiset → Kuntotarkistus
3. **Suorita:** Ylläpito → Liitännäiset → Etsi tallentamatta · `dryRun=true`
4. **Vertaa:** `/api/next/digital-items?limit=200`
