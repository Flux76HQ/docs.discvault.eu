## Toimenpiteet

1. **Tarkista:** Docker Engine `≥24` · Docker Compose `v2`
2. **Tarkista:** `≥2 GB` · kirjoitettava pysyvä tallennus ja vapaa varmuuskopiotila
3. **Määritä:** DNS · TLS · HTTPS · `RP_ID` · `RP_ORIGINS`
4. **Kirjaa:** valittu julkaisukanava
